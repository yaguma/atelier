# ローグライクテンプレート

不思議のダンジョン、Hades、Enter the Gungeonなどのローグライク/ローグライト向け。

## 典型的なプレイサイクル

```
① 状況提示: 部屋/フロアの状態表示、敵配置、アイテム配置
② プレイヤーの選択: 移動/攻撃/アイテム使用/スキル発動
③ 結果反映: 敵撃破/被ダメージ/アイテム獲得/フロア進行
④ フィードバック: エフェクト、ドロップ演出、進行表示
→ ①に戻る
```

## ジャンル特有の要素

### パーマデス
- 死亡時の扱い（全ロスト/一部持ち越し）
- メタ進行（永続アンロック）

### 手続き型生成
- マップ生成方式
- 敵配置ルール
- アイテム出現ルール

### ビルド構築
- スキル/武器の選択
- シナジー要素
- アップグレードシステム

### 進行構造
- フロア構成
- ボス配置
- ショップ/イベントの頻度

## ヒアリング質問

### パーマデスについて
- 「死んだら最初から？」
- 「持ち越せるものはある？」
- 「難易度調整はある？」

### 生成について
- 「マップはランダム生成？」
- 「敵配置のルールは？」

### ビルドについて
- 「武器/スキルはいくつ持てる？」
- 「成長要素は？」
- 「シナジー（組み合わせボーナス）はある？」

## 出力テンプレート

```markdown
# {ゲーム名} 要件定義書（ローグライク）

## 1. プレイサイクル

### 探索ループ（30秒〜1分）
① 状況提示: 部屋の構造と敵/アイテム表示
② プレイヤーの選択: 移動先、攻撃対象、アイテム使用
③ 結果反映: 戦闘結果、アイテム獲得、部屋クリア
④ フィードバック: ダメージ表示、クリア演出

### ラン全体（15〜30分）
- フロア攻略 → 報酬選択 → 次フロアへ → ボス戦

## 2. 勝敗条件
- 勝利条件: 最終ボスを撃破
- 敗北条件: HPが0になる（パーマデス）
- 撤退条件: {あり/なし}

## 3. プレイヤー操作
- 移動する
- 攻撃する
- 回避する
- アイテムを使う
- スキルを発動する
- 装備を変更する

## 4. ゲーム要素

### プレイヤー
- HP: {N}
- 武器スロット: {N}個
- スキルスロット: {N}個
- 所持アイテム: {N}個まで

### 武器
- 種類: 近接/遠距離/魔法
- レアリティ: コモン/レア/レジェンダリー
- 特殊効果: {効果リスト}

### 敵
- 種類: 雑魚/エリート/ボス
- 行動パターン: {パターンリスト}

### マップ
- フロア数: {N}
- 部屋構成: 戦闘/ショップ/イベント/ボス
- 分岐: {あり/なし}

### メタ進行
- 永続アンロック: {アンロック要素}
- 通貨: {通貨名}

## 5. 数値設計

| 変数 | 範囲 | 初期値 |
|-----|------|-------|
| プレイヤーHP | 100〜200 | 100 |
| フロア数 | 5〜10 | 5 |
| 部屋/フロア | 5〜15 | 8 |
| 武器スロット | 1〜2 | 1 |
| スキルスロット | 2〜4 | 3 |

## 6. 想定プレイフロー

### フロア1（序盤）
- 状況: 初期装備、HP100
- 行動: 雑魚3体と戦闘、宝箱から武器獲得
- 結果: HP85、新武器装備

### フロア3（中盤）
- 状況: 強化済み装備、HP60、スキル3つ習得
- 行動: エリート敵と戦闘
- 結果: HP30、レア装備獲得

### 最終フロア
- 状況: フルビルド完成、HP50
- 行動: ボス戦
- 結果: 撃破 or 敗北
```
