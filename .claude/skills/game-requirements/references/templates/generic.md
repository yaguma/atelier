# 汎用テンプレート

ジャンルが特定できない場合に使用する汎用テンプレート。

## プレイサイクル例

```
① 状況提示: ゲーム状態が表示される
② プレイヤーの選択: 利用可能な選択肢から1つ選ぶ
③ 結果反映: 選択に応じてゲーム状態が変化
④ フィードバック: 結果が視覚・聴覚で伝わる
→ ①に戻る
```

## ヒアリングガイド

### まず聞くこと
1. 「どんなゲームをイメージしている？」
2. 「参考にしているゲームはある？」
3. 「1プレイの長さは？（5分？30分？1時間？）」

### 次に聞くこと
1. 「プレイヤーは何を繰り返す？」
2. 「何が楽しい？どこが気持ちいい？」
3. 「どうなったら勝ち？負け？」
