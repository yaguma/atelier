# BGM プレースホルダー

このフォルダにはゲームで使用するBGMファイルを配置します。

## 必要なファイル

以下のIDに対応するAudioClipを配置してください（.wav, .mp3, .ogg形式）:

| ファイル名 | 説明 | 使用場面 |
|-----------|------|---------|
| BGM_TITLE | タイトルBGM | タイトル画面 |
| BGM_MAP | マップBGM | マップ/ノード選択画面 |
| BGM_QUEST | クエストBGM | 調合画面 |
| BGM_MERCHANT | 商人BGM | 商人画面 |
| BGM_RESULT_WIN | 勝利BGM | 結果画面（成功） |
| BGM_RESULT_LOSE | 敗北BGM | 結果画面（失敗） |

## 音源仕様

- サンプルレート: 44100Hz推奨
- ビット深度: 16bit推奨
- チャンネル: ステレオ
- ループ対応: 各BGMはループ再生されます

## プレースホルダー音源

開発中は無音のAudioClipをプレースホルダーとして使用することができます。
