# SE プレースホルダー

このフォルダにはゲームで使用するSE（効果音）ファイルを配置します。

## 必要なファイル

以下のIDに対応するAudioClipを配置してください（.wav, .mp3, .ogg形式）:

### UI系SE

| ファイル名 | 説明 | 使用場面 |
|-----------|------|---------|
| SE_BUTTON_HOVER | ボタンホバーSE | ボタンにカーソルが乗った時 |
| SE_BUTTON_CLICK | ボタンクリックSE | ボタン押下時 |
| SE_DIALOG_OPEN | ダイアログ表示SE | ダイアログが開いた時 |
| SE_DIALOG_CLOSE | ダイアログ閉じるSE | ダイアログが閉じた時 |
| SE_CONFIRM | 決定SE | 決定操作時 |
| SE_CANCEL | キャンセルSE | キャンセル操作時 |
| SE_CURSOR | カーソル移動SE | メニューカーソル移動時 |

### ゲーム系SE

| ファイル名 | 説明 | 使用場面 |
|-----------|------|---------|
| SE_CARD_PLAY | カードプレイSE | カードを使用した時 |
| SE_CARD_DRAW | カードドローSE | カードを引いた時 |
| SE_TURN_END | ターン終了SE | ターン終了時 |
| SE_BREW_SUCCESS | 調合成功SE | 調合成功時 |
| SE_BREW_FAIL | 調合失敗SE | 調合失敗時 |
| SE_EXPLOSION | 暴発SE | 調合暴発時 |
| SE_QUEST_COMPLETE | クエスト完了SE | クエストクリア時 |
| SE_GOLD_GAIN | ゴールド獲得SE | ゴールド入手時 |
| SE_FAME_GAIN | 名声獲得SE | 名声入手時 |

## 音源仕様

- サンプルレート: 44100Hz推奨
- ビット深度: 16bit推奨
- チャンネル: モノラルまたはステレオ
- 長さ: 通常0.1秒〜2秒程度

## プレースホルダー音源

開発中は無音のAudioClipをプレースホルダーとして使用することができます。
