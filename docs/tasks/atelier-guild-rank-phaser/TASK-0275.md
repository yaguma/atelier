## 確認手順

### セットアップ

1. 開発サーバーを起動: `npm run dev`
2. ブラウザでアクセス: `http://localhost:5173`
3. Chrome DevToolsを開く（F12）
4. Consoleタブでエラーを監視

### 確認フロー

```
1. タイトル画面
   ↓ 新規ゲーム
2. 依頼受注フェーズ
   ↓ 依頼を2件受注してスキップ
3. 採取フェーズ
   ↓ 全APを使用して採取
4. 調合フェーズ
   ↓ 可能であれば1品調合
5. 完了
```
