### 2. アセットパスユーティリティ 🔴

**信頼性**: 🔴 *ディレクトリ構成は調整が必要*

**実装ファイル**: `src/presentation/phaser/config/AssetPaths.ts`

```typescript
const BASE_PATH = 'assets';

export const AssetPaths = {
  image: (key: string) => `${BASE_PATH}/images/${key}.png`,
  audio: (key: string) => `${BASE_PATH}/audio/${key}.mp3`,
  font: (key: string) => `${BASE_PATH}/fonts/${key}.ttf`,
  json: (key: string) => `${BASE_PATH}/data/${key}.json`,
} as const;
```

### 3. アセットマニフェスト 🔴

**信頼性**: 🔴 *実際のアセット準備後に調整*

**実装ファイル**: `src/presentation/phaser/config/AssetManifest.ts`

```typescript
import { ImageKeys, AudioKeys } from './AssetKeys';
import { AssetPaths } from './AssetPaths';

export interface AssetEntry {
  key: string;
  path: string;
}

export const ImageManifest: AssetEntry[] = Object.values(ImageKeys).map(key => ({
  key,
  path: AssetPaths.image(key)
}));

export const AudioManifest: AssetEntry[] = Object.values(AudioKeys).map(key => ({
  key,
  path: AssetPaths.audio(key)
}));
```
