# 🧭 ゲーム企画ノート：『◯◯の錬金術アトリエ（仮）』

## 🏷️ プロジェクト概要

**ジャンル**

> 錬金術 × 短期ローグライク × デッキ構築

**開発方針**

> 20〜40分で遊べる「戦略パズル×爽快ループ」体験を目指す。
> 毎プレイ異なる素材・顧客・デッキ構成でリプレイ性を重視。

**コンセプトコピー（仮）**

> 「混ぜて、暴発して、満足させろ。」

---

## 🎮 ゲーム概要

**🎯 プレイヤーの目的**

* 顧客の依頼を達成し、名声を上げながら錬金工房を発展させる。
* 1プレイ内で最終依頼（王家など）を達成する。

**🕒 プレイ時間**

* 1プレイ＝20〜40分
* 1周ごとに素材・顧客・カード構成が変化。

---

## ⚗️ ゲームフロー

| フェーズ        | 内容                 |
| ----------- | ------------------ |
| 1️⃣ スタート    | 流派（錬金スタイル）選択       |
| 2️⃣ フィールド進行 | ノード（依頼／商人／実験／魔物）選択 |

| 5️⃣ 最終依頼    | 高難度レシピ制作に挑戦        |

## 🗺️ マップ構造（ノード進行）

「Slay the Spire」式のマップで進行。
ルート選択によってカード獲得やイベントが変化。

| ノード      | 内容                    |
| -------- | --------------------- |
| 👩‍🌾 依頼 | 顧客の要求を満たす戦闘フェーズ       |
| 🏪 商人    | 素材・触媒カードを購入／強化        |
| 🧪 実験    | ランダムイベント（成功で進化、失敗で暴発） |
| 🐉 魔物    | 素材を入手する戦闘             |
| 👑 ボス依頼  | 高難度顧客。突破で次エリアへ        |

---

## ⚔️ 顧客（戦闘）フェーズ

**🎯 勝利条件**

* 指定属性値・品質・安定性の条件達成
  　例：「回復＝水3＋光2」
* または満足度ゲージを満たす

**💥 失敗ペナルティ**

* 名声減少
* デッキ1枚ロスト（素材枯渇）
* 次ルートで顧客数減少

---

## 🧩 カードシステム概要

各カードは「属性値」「安定値」「反応効果」を持つ。

| カード例 | 属性値     | 安定 | 効果       |
| ---- | ------- | -- | -------- |
| 薬草   | 水+1     | +1 | 回復系が安定化  |
| 火薬   | 火+2     | -1 | 爆発リスク上昇  |
| 魔石   | 光+1／闇+1 | 0  | ランダム効果発生 |
| 蒸留   | 変換（火→水） | +1 | 精製ボーナス   |

## ⚙️ 戦闘システム：「複数依頼制」

同時に最大3件の顧客依頼を受け、
**手札をどの依頼に割り当てるか**で戦略が変わる。

| 顧客  | 条件     |
| --- | ------ |
| 治癒師 | 安定5以上  |
| 冒険者 | 暴発1回発生 |
| 貴族  | 品質8以上  |

→ 手札状況で依頼を選び、リソースを最適化。

### 管理モード案

| モード      | 内容                | 特徴       |
| -------- | ----------------- | -------- |
| 🧾 並行依頼制 | 各ターンどの依頼に割り当てるか選択 | 戦略性高     |
| 🔁 キュー制  | 1件完了で次が出現         | テンポ重視    |
| ⏳ 期限制    | 各依頼にターン制限あり       | 緊張感・時間管理 |

---

## 💥 依頼間シナジー（反応連鎖）

| 現象   | 効果              |
| ---- | --------------- |
| 共鳴反応 | 同属性依頼を同時進行でボーナス |
| 廃液転用 | 副産物を他依頼で再利用     |
| 暴発連鎖 | 一方の暴発が他依頼に影響    |
| 同時納品 | 2件同時達成で名声ボーナス   |
