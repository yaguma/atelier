# アトリエ錬金術ゲーム 要件定義書（ギルドランク制）

**バージョン**: 1.7
**作成日**: 2025-12-30
**更新日**: 2026-02-23
**形式**: game-requirementsスキル準拠

---

## 概要

本ドキュメントは、錬金術をテーマにしたギルドランク制デッキ構築RPGの要件定義書である。
依頼を「戦闘的な要素」として扱い、レシピから調合を行うことで依頼を達成し、ギルドランクを上げていくゲーム。

**ジャンル**: 錬金術 × デッキ構築 × ギルドランク制RPG
**コンセプト**: 「依頼を受けて、素材を採って、調合して、ランクを上げろ。」
**プレイ時間**: 1ランク攻略 10〜20分 / 全ランク攻略 60〜90分

### ワンセンテンス・コンセプト

> 錬金術師（プレイヤー）が、採取地カードとレシピカードのデッキを駆使し、
> 依頼を達成して貢献度を稼ぎ、ギルドランクという「敵」を倒していく
> 「戦略的デッキ構築」と「ランクアップの達成感」を味わうゲーム

### 中核の面白さ

> このゲームは「採取地とレシピのバランスを考えてデッキを組み、
> 限られた素材と期限の中で依頼を達成し、ギルドランクを攻略していくことが楽しい」

## 1. プレイサイクル

### 1.1 全体構造（ギルドランク制）

```
Gランク → Fランク → Eランク → Dランク → Cランク → Bランク → Aランク → Sランク
   ↓         ↓         ↓         ↓         ↓         ↓         ↓         ↓
 [昇格ゲージを溜める → 昇格試験クリア → 次ランクへ]
 [ランクアップで採取地・レシピ・依頼が解放]
```

- **1ランク攻略時間**: 10〜20分
- **全ランク攻略時間**: 60〜90分
- **ランクごとに難易度上昇**: 特殊ルール追加、要求品質上昇

### 1.2 ランク内のメインループ（1日: 30秒〜1分）

```
┌─────────────────────────────────────────────────────────┐
│                    1日の流れ                            │
├─────────────────────────────────────────────────────────┤
│ ① 依頼受注フェーズ                                      │
│    - 依頼者が来訪（1〜3人）                             │
│    - 依頼内容を確認し、受注するか選択                    │
│    - 同時受注上限: 3件                                  │
│    - コスト: 0（受注自体は無料）                        │
├─────────────────────────────────────────────────────────┤
│ ② 採取フェーズ                                          │
│    - 手札から採取地カードを使ってドラフト採取開始        │
│    - 3つの素材が提示され、1つ選んで獲得（繰り返し可能）  │
│    - 採取地の距離＋選択個数に応じて行動ポイント消費      │
│    - スキップ可能                                       │
├─────────────────────────────────────────────────────────┤
│ ③ 調合フェーズ                                          │
│    - 手札からレシピカードを使ってアイテムを調合          │
│    - 行動ポイントを消費（1〜3）                         │
│    - スキップ可能                                       │
├─────────────────────────────────────────────────────────┤
│ ④ 納品フェーズ                                          │
│    - 完成したアイテムを依頼者に納品                      │
│    - 貢献度を獲得 → 昇格ゲージが上昇                    │
│    - 報酬カード選択: 3枚の中から1枚を獲得               │
│    - コスト: 0（納品自体は無料）                        │
├─────────────────────────────────────────────────────────┤
│ 日終了処理                                              │
│    - 依頼期限-1                                         │
│    - 手札調整（5枚になるまで補充）                      │
│    - ランク判定（昇格ゲージ満タンで昇格試験解放）       │
└─────────────────────────────────────────────────────────┘
```

**行動ポイント**: 3/日（採取・調合フェーズで共有）
**買い物**: どのフェーズでも可能（1コスト）

### 1.3 フェーズ別行動コスト

| フェーズ | 行動 | コスト | 詳細 |
|---------|------|--------|------|
| **依頼受注** | 依頼を受注 | 0 | 依頼者から依頼を受ける |
| **依頼受注** | 依頼を断る | 0 | 依頼者の依頼を断る |
| **採取** | 採取地カード使用 | 基本+追加 | 下記「採取コスト計算」参照 |
| **調合** | レシピカード使用 | 1〜3 | レシピの複雑さによる |
| **納品** | 依頼を納品 | 0 | 完成品を依頼者に納品 |
| **共通** | 買い物 | 1 | どのフェーズでも可能 |
| **共通** | 休憩 | 0 | 手札を2枚捨てて2枚ドロー |
| **共通** | 確認系 | 0 | ステータス確認など |

#### 採取コスト計算

**採取コスト = 基本コスト（採取地の距離） + 追加コスト（選択個数）**

| 選択個数 | 追加コスト | 備考 |
|---------|-----------|------|
| 0個（偵察のみ） | 0 | 何も採らない場合 |
| 1〜2個 | 1 | 軽い採取 |
| 3〜4個 | 2 | 普通の採取 |
| 5〜6個 | 3 | 重い採取 |
| 7個以上 | 4 | 大量採取 |

> **v1.7変更**: 旧「7個以上で+1日」ルールをAP超過モデルに統合。追加日数カラムを廃止し、APコスト4に変更。
> 日数消費はAP超過計算アルゴリズムで統一管理される。
> 詳細は [フェーズ自由遷移要件定義書 REQ-003-07](free-phase-navigation/requirements.md) を参照。

※基本コストは採取地カードごとに設定（詳細は4.2参照）
※AP消費の上限はMAX_AP×2=6（REQ-003-08参照）

## 2. 勝敗条件

| 条件 | 内容 |
|------|------|
| **勝利条件** | Sランク到達（全ランク攻略） |
| **ランクアップ条件** | 制限日数内に昇格ゲージを満タンにする → 昇格試験クリア |
| **敗北条件** | 制限日数を超過 = **ゲームオーバー（完全リセット）** |

### 2.1 日数制限システム

#### 基本ルール
- 各ランクに制限日数が設定される
- 制限日数内に昇格ゲージを満タンにできないと即ゲームオーバー
- 昇格試験クリア時に日数がリセットされる
- 依頼失敗のペナルティは無し（日数の浪費自体が実質ペナルティ）

#### ランクごとの制限日数

| ランク | 必要貢献度 | 制限日数 | 難易度感 |
|--------|-----|----------|---------|
| G | 100 | 30日 | チュートリアル、余裕あり |
| F | 200 | 30日 | まだ余裕あり |
| E | 350 | 35日 | 少し意識が必要 |
| D | 500 | 35日 | 効率的なプレイ必要 |
| C | 700 | 35日 | タイトになってくる |
| B | 1000 | 35日 | 高効率必須 |
| A | 1500 | 35日 | 極めてタイト |
| S | —（最終ランク） | 無制限 | エンドコンテンツ（自由プレイ） |

> **Sランクについて**: Sランクは最終ランクのため昇格はない。A→S昇格試験クリア=ゲームクリア。
> Sランク到達後は制限日数なしの自由プレイモードとなり、チャレンジ依頼や称号収集を楽しめる。
> 詳細は「4.7 ギルドランクシステム」のSランク詳細を参照。

## 3. プレイヤー操作

### 3.1 依頼受注フェーズの操作

| 操作 | 詳細 |
|------|------|
| **依頼者を確認する** | 今日来訪した依頼者（1〜3人）を確認 |
| **依頼内容を見る** | 依頼者の要望、報酬、期限を確認 |
| **依頼を受ける** | 依頼を受注（最大3件同時受注可能） |
| **依頼を断る** | 今日の依頼者の依頼を断る |
| **受注中の依頼を確認** | 現在受注している依頼一覧を確認 |
| **依頼を破棄する** | 受注中の依頼を破棄（ペナルティなし） |

### 3.2 採取フェーズの操作

| 操作 | 詳細 |
|------|------|
| **採取地カードを使う** | 手札から採取地を選んでドラフト採取を開始 |
| **素材を選ぶ** | 提示された3つの素材から1つを選んで獲得 |
| **採取を続ける** | 次の素材提示へ進む（提示回数上限まで繰り返し可能） |
| **採取を終える** | ドラフト採取を終了し、選択個数に応じたコストを消費 |
| **素材を確認する** | インベントリの素材を確認 |
| **フェーズをスキップ** | 採取せずに調合フェーズへ進む |

#### ドラフト採取の流れ

```
1. 採取地カードを選択
   ↓
2. 素材プールから3つの素材がランダム提示
   ↓
3. 1つ選ぶ or スキップ（何も選ばない）
   ↓
4. 「続ける」→ 2に戻る（提示回数上限まで）
   「終わる」→ 採取終了、コスト計算
   ↓
5. 基本コスト＋追加コストを消費して素材獲得
```

### 3.3 調合フェーズの操作

| 操作 | 詳細 |
|------|------|
| **レシピカードを使う** | 手札からレシピを選んで調合開始（行動ポイント消費） |
| **素材をセットする** | 必要素材をインベントリから選択 |
| **強化カードを使う** | 調合と同時に使用して品質アップなど |
| **調合する** | アイテムを完成させる |
| **フェーズをスキップ** | 調合せずに納品フェーズへ進む |

### 3.4 納品フェーズの操作

| 操作 | 詳細 |
|------|------|
| **納品する** | 完成品を依頼者に納品 → 貢献度獲得 |
| **報酬カードを選ぶ** | 納品成功時、3枚の候補から1枚を選んでデッキに追加 |
| **強化カードを使う** | 納品と同時に使用して貢献度アップなど |
| **納品をスキップ** | 今日は納品せずに日終了へ |

#### 報酬カード選択のルール

- 依頼を1件納品するごとに、3枚の報酬カード候補が提示される
- 候補は依頼のタイプ・難易度・依頼者によって変化
- 選んだカードは即座にデッキに追加される
- 「選ばない」ことも可能（デッキ圧縮戦略）

### 3.5 共通操作（どのフェーズでも可能）

| 操作 | コスト | 詳細 |
|------|--------|------|
| **買い物する** | 1 | ショップでカードや素材を購入 |
| **休憩する** | 0 | 手札を2枚捨てて2枚ドロー |
| **手札を確認する** | 0 | 現在の手札を確認 |
| **デッキを確認する** | 0 | 全カードを確認 |
| **ランク進捗を見る** | 0 | 現在のランクとHP残量を確認 |
| **残り日数を確認する** | 0 | 現在のランクの残り日数を確認 |
| **貢献度を確認する** | 0 | 累計貢献度を確認 |

## 4. ゲーム要素

### 4.1 カード系統（3系統）

| 系統 | 役割 | 効果例 |
|------|------|--------|
| **採取地カード** | ドラフト採取で素材を獲得 | 近くの森: 3回提示、薬草など |
| **レシピカード** | 素材を消費して調合 | 回復薬: 薬草×2+清水×1→回復薬 |
| **強化カード** | 行動を強化するサポート | 賢者の触媒: 調合品質+1ランク |

### 4.2 採取地カードの設計（ドラフト採取方式）

採取地カードを使用すると、ドラフト形式で素材を選んで獲得できる。

#### 採取地カード一覧

| 採取地 | 基本コスト | 提示回数 | 素材プール | レア出現率 |
|--------|-----------|---------|-----------|-----------|
| **裏庭** | 0 | 2回 | 雑草、水 | 0% |
| **近くの森** | 0 | 3回 | 薬草、キノコ、木材、清水 | 10% |
| **川辺** | 0 | 3回 | 魚、水草、砂、清水 | 10% |
| **山麓の岩場** | 1 | 4回 | 鉱石、石、レア鉱石 | 15% |
| **奥地の洞窟** | 1 | 4回 | レア苔、鉱石、魔法素材 | 20% |
| **火山地帯** | 2 | 5回 | 火山石、灰、溶岩石 | 25% |
| **古代遺跡** | 2 | 5回 | 魔法素材、古代の欠片 | 30% |

#### ドラフト採取のルール

| 要素 | 内容 |
|------|------|
| **1回の提示** | 素材プールから3つをランダム提示 |
| **選択** | 1つ選んで獲得 or スキップ（何も選ばない） |
| **提示回数** | 採取地により異なる（上限まで繰り返し可能） |
| **品質** | 提示される素材には品質（D〜S）がランダムで付与 |

#### 採取コスト計算（再掲）

**採取コスト = 基本コスト（採取地の距離） + 追加コスト（選択個数）**

| 選択個数 | 追加コスト |
|---------|-----------|
| 0個 | 0 |
| 1〜2個 | 1 |
| 3〜4個 | 2 |
| 5〜6個 | 3 |
| 7個以上 | 4 |

> ※日数消費はAP超過計算アルゴリズムで統一管理（v1.7変更）

#### 採取の例

```
【近くの森で採取】（基本コスト0、提示回数3回）

1回目: [薬草(C)] [キノコ(C)] [木材(D)] → 薬草を選択
2回目: [清水(C)] [薬草(B)] [キノコ(D)] → 薬草(B)を選択
3回目: [木材(C)] [清水(C)] [薬草(C)] → 清水を選択

獲得: 薬草(C)、薬草(B)、清水(C)
コスト: 基本0 + 追加2（3個選択） = 2コスト消費
```

### 4.3 レシピカードの設計

| レシピ | コスト | 必要素材 | 対応依頼カテゴリ |
|--------|--------|---------|-----------------|
| **回復薬** | 1 | 薬草×2、清水×1 | 医療系 |
| **解毒剤** | 1 | 毒キノコ×1、清水×2 | 医療系 |
| **栄養剤** | 1 | 薬草×1、魚×1、水×1 | 医療系 |
| **爆弾** | 2 | 火山石×1、硫黄×1、油×1 | 冒険者向け |
| **鋼の剣** | 2 | 鉱石×3、木材×1 | 武具系 |
| **魔法の杖** | 2 | 魔法素材×2、木材×1 | 魔法系 |
| **万能薬** | 3 | 高山薬草×2、聖水×1、月の雫×1 | 高級医療系 |

### 4.4 強化カードの設計

#### 基本ルール
- **コスト**: 0（他のカードと併用して使用）
- **使用タイミング**: 対象の行動と同時に使用
- **効果**: その行動を1回だけ強化

#### 強化カード一覧

| カード名 | 効果 | 対象行動 | 入手方法 |
|---------|------|---------|---------|
| **賢者の触媒** | 調合品質+1ランク | 調合 | 初期デッキ、ショップ |
| **錬金の灰** | 素材を1つ節約 | 調合 | ショップ |
| **精霊の導き** | 提示回数+1回 | 採取 | 初期デッキ、ショップ |
| **幸運のお守り** | レア素材出現確率+30% | 採取 | ショップ |
| **交渉術の書** | 報酬金+50% | 納品 | ショップ |
| **ギルド推薦状** | 貢献度+30% | 納品 | 初期デッキ、ショップ |
| **集中の瞑想** | 次の行動コスト-1 | 全行動 | ショップ（レア） |

### 4.5 素材システム（消費制）

#### 基本ルール
- 採取で獲得、調合で**永続消費**
- 品質: D < C < B < A < S
- 属性: 火・水・土・風
- **保管上限**: 20枠（拡張可能）

#### 素材例

| 素材 | 品質 | 属性 | 入手場所 |
|------|------|------|---------|
| 薬草 | C | 草・水 | 近くの森 |
| 高山薬草 | A | 草・水 | 遠くの山 |
| 鉱石 | C | 火・土 | 山麓の岩場 |
| 火山石 | A | 火 | 火山地帯 |
| 魔法素材 | S | 全属性 | 古代遺跡 |

### 4.6 依頼システム（攻撃手段）

#### 依頼の基本構造

| 要素 | 詳細 |
|------|------|
| **依頼者** | 依頼を出すキャラクター（詳細は4.11参照） |
| **要望（依頼タイプ）** | 8種類の条件から1つ（詳細は4.12参照） |
| **報酬** | 貢献度 + お金 + 報酬カード選択 |
| **期限** | 受注から一定日数以内 |
| **フレーバーテキスト** | 「〜が欲しいんだ」などの依頼者のセリフ |

#### 報酬カード選択システム

納品成功時、3枚のカード候補が提示され、1枚を選んでデッキに追加できる。

##### カード候補の決定ロジック

```
【カード候補プール】
1. 依頼者タイプに関連するカード（1枚）
   - 村人: 基本採取地カード、基本レシピカード
   - 冒険者: 武器系レシピ、遠方の採取地
   - 商人: 強化カード、経済系カード
   - 貴族: 高級レシピ、レア強化カード
   - ギルド: 任意（ランダム）

2. 依頼タイプに関連するカード（1枚）
   - 具体的指定: その依頼で納品したアイテムのレシピ強化版
   - カテゴリ: そのカテゴリの別レシピ
   - 品質条件: 品質上昇系の強化カード
   - 数量重視: 素材獲得量アップ系カード
   - 属性条件: その属性の素材が取れる採取地
   - 効果ベース: 効果強化系カード
   - 素材消費: レア素材採取地
   - 複合条件: ランダム高レアカード

3. ランダム枠（1枚）
   - 現在のランクで入手可能なカードからランダム
```

##### 難易度別の報酬カードレアリティ

| 依頼難易度 | コモン | アンコモン | レア |
|-----------|--------|----------|------|
| 簡単 | 80% | 18% | 2% |
| 普通 | 60% | 35% | 5% |
| 難しい | 40% | 45% | 15% |
| 最難関 | 20% | 50% | 30% |

##### 特別ボーナス

- **品質S納品**: レア枠が確定で1枚追加（計4枚から選択）
- **期限-1日で納品**: アンコモン以上確定
- **コンボ5連続以上**: 選択肢が1枚増加

#### 依頼タイプ一覧（8種類）

| タイプ | 説明 | 例 | 解放ランク |
|--------|------|-----|----------|
| **具体的指定** | 特定アイテムを指定 | 「回復薬が欲しい」 | G（初期） |
| **カテゴリ** | カテゴリで指定 | 「何か薬が欲しい」 | G（初期） |
| **品質条件** | 品質値で指定 | 「品質60以上のアイテム」 | F |
| **数量重視** | 数量で指定 | 「薬を5個欲しい」 | F |
| **属性条件** | 属性値で指定 | 「火属性10以上の武器」 | E |
| **効果ベース** | 効果で指定 | 「HP50回復できるもの」 | E |
| **素材消費** | 使用素材で指定 | 「レア素材を使った品」 | D |
| **複合条件** | 複数条件を組み合わせ | 「火属性+品質B以上」 | C |

#### 貢献度計算

```
最終貢献度 = 基本貢献度 × 品質補正 × 依頼タイプ補正 × コンボ補正

【品質補正】
  D: ×0.5
  C: ×1.0
  B: ×1.5
  A: ×2.0
  S: ×3.0

【依頼タイプ補正】
  具体的指定: ×1.0
  カテゴリ: ×0.8
  品質条件: ×1.2
  数量重視: ×0.7（1個あたり）
  属性条件: ×1.3
  効果ベース: ×1.3
  素材消費: ×1.5
  複合条件: ×1.8

【コンボ補正】（連続依頼達成）
  2連続: ×1.1
  3連続: ×1.2
  5連続: ×1.5
  10連続: ×2.0（フィーバー！）
```

### 4.7 ギルドランクシステム（敵）

#### ランク一覧

| ランク | 名称 | 必要貢献度 | 制限日数 | 特殊ルール | 解放要素 |
|--------|------|-----|----------|-----------|---------|
| G | 見習い | 100 | 30日 | なし（チュートリアル） | 基本採取地、基本レシピ |
| F | 新人 | 200 | 30日 | なし | 川辺、解毒剤 |
| E | 一人前 | 350 | 35日 | 品質D以下は貢献度半減 | 山麓、爆弾 |
| D | 中堅 | 500 | 35日 | 全依頼の期限-1日 | 洞窟、鋼の剣 |
| C | 熟練 | 700 | 35日 | 品質C以上でないと受理されない | 火山、魔法系レシピ |
| B | 上級 | 1000 | 35日 | 品質B以上必須、期限-1日 | 遺跡、高級レシピ |
| A | 最上級 | 1500 | 35日 | 品質A以上必須、期限-2日 | 全採取地、秘伝レシピ |
| S | 伝説 | —（最終ランク） | 無制限 | 全制限解除、チャレンジ依頼出現 | 称号、全レシピ解放、チャレンジ依頼 |

> **Fランクの同時受注制限について（設計意図）**:
> Gランク（チュートリアル）では同時受注3件が可能。Fランクで同時受注を制限すると
> ランクアップが「パワーダウン」に感じられるため、Fランクには同時受注制限を設けない。
> 同時受注上限は全ランク共通で3件とする。難易度上昇はE以降の品質制限・期限短縮で表現する。

#### Sランク詳細（エンドコンテンツ）

Sランクは最終ランクであり、A→S昇格試験「伝説のアイテム製作」をクリアすることでゲームクリアとなる。
Sランク到達後は以下のエンドコンテンツが解放される。

| 要素 | 内容 |
|------|------|
| **制限日数** | 無制限（自由プレイ） |
| **チャレンジ依頼** | 高難度の特別依頼。通常依頼とは別枠で掲示板に出現し、達成で称号を獲得 |
| **全レシピ解放** | 全ランクのレシピが購入可能になる |
| **全採取地解放** | 全ての採取地にアクセス可能 |
| **称号システム** | チャレンジ依頼クリア・アイテムコンプリート等の実績で称号を獲得 |
| **品質制限** | なし（全品質の納品が可能） |

> **エンドコンテンツの設計方針**: Sランクは「やり込み要素」の提供が目的。
> 制限日数がないため失敗リスクはなく、プレイヤーは自分のペースで
> チャレンジ依頼や称号収集、全アイテムコンプリートを目指せる。
> 将来的にリーダーボード（最少日数クリア等）の追加も検討可能。

#### 昇格試験

各ランクの昇格ゲージ満タン到達後、昇格試験が解放される。

| 試験 | 課題内容 | 制限 |
|------|---------|------|
| G→F | 回復薬×2 納品 | 5日以内 |
| F→E | 回復薬(B以上)×3、解毒剤×2 | 5日以内 |
| E→D | 爆弾×2、鋼の剣×1 | 4日以内 |
| D→C | 指定アイテム3種を同時納品 | 4日以内 |
| C→B | 全品質B以上で5件達成 | 3日以内 |
| B→A | 複合レシピ3種納品 | 3日以内 |
| A→S | 伝説のアイテム製作 | 制限なし |

#### 昇格報酬

昇格試験クリア時に以下の報酬を獲得：
- **アーティファクト選択**: 3つの候補から1つを選択して獲得
- **ボーナスゴールド**: ランクに応じた報酬金（50G〜300G）
- **新規カード解放**: 次のランクで購入可能なカードが解放

### 4.8 日数管理（緊張感）

#### 基本ルール
- 各ランクには制限日数があり、画面上に残り日数が表示される
- 1日の終了時に残り日数が1減少する
- 残り日数が0になった時点で昇格ゲージが満タンでないと**ゲームオーバー**
- 昇格試験をクリアすると、次のランクの制限日数にリセットされる

#### 緊張感を生むデザイン
- **残り10日以下**: 警告表示（黄色）
- **残り5日以下**: 危険表示（赤色）+ 警告音
- **残り日数は常に画面に表示**: プレイヤーは常に意識せざるを得ない

#### 依頼失敗について
- 依頼期限切れ（失敗）時のペナルティは**なし**
- ただし、日数を浪費したこと自体が間接的なペナルティとなる
- シンプルなルールで緊張感を維持

### 4.11 依頼者システム

#### 概要

依頼者は依頼を持ってくるキャラクターであり、毎日1〜3人が工房を訪れる。
依頼者のタイプによって依頼内容の傾向と報酬が異なる。

#### 依頼者タイプ

| タイプ | 特徴 | 依頼傾向 | 報酬傾向 | 登場ランク |
|--------|------|---------|---------|----------|
| **村人** | 日常的な依頼 | カテゴリ・数量系 | 低報酬・長期限 | G（初期） |
| **冒険者** | 武器・爆弾系 | 属性・効果系 | 中報酬・中期限 | G（初期） |
| **商人** | 品質重視 | 品質・複合系 | 高報酬・短期限 | E |
| **貴族** | 高品質・レア品 | 複合・素材消費系 | 最高報酬・最短期限 | C |
| **ギルド** | 昇格試験用 | 具体的指定 | 貢献度ボーナス | G（特別） |

#### 依頼者の登場ロジック

```
【毎日の依頼受注フェーズ】
1. 依頼者プールからランダムに1〜3人を抽選
2. 各依頼者が依頼タイプに応じた依頼を1件持参
3. プレイヤーは依頼を受けるか断るかを選択

【依頼者プール】
- ランクに応じて登場する依頼者タイプが増加
- 高ランク依頼者ほど出現確率が低い
```

|--------|----------|---------|---------|

| 冒険者 | ×1.0 | ×1.0 | ±0日 |
| 商人 | ×1.2 | ×1.5 | -1日 |
| 貴族 | ×1.5 | ×2.0 | -2日 |

### 4.12 依頼タイプ詳細

各依頼タイプの判定ロジックと具体例。

#### 具体的指定

```
条件: アイテムID = 指定アイテム
判定: 完成品のアイテムIDが一致するか
例:
  - 「回復薬が欲しい」→ 回復薬を調合して納品
  - 「爆弾が2個必要だ」→ 爆弾×2を納品
```

#### カテゴリ

```
条件: カテゴリ = 指定カテゴリ
判定: 完成品のカテゴリが一致するか
例:
  - 「何か薬が欲しい」→ 回復薬/解毒剤/栄養剤のいずれかでOK
  - 「武器なら何でも」→ 鋼の剣/魔法の杖などでOK
```

#### 品質条件

```
条件: 品質 >= 指定値
判定: 完成品の品質値が条件を満たすか
例:
  - 「品質60以上のアイテム」→ 品質60以上なら何でもOK
  - 「品質B以上の薬」→ 品質B以上の薬カテゴリ
```

#### 数量重視

```
条件: カテゴリ = 指定カテゴリ AND 合計数量 >= 指定数
判定: 条件を満たすアイテムの合計数
例:
  - 「薬を5個欲しい」→ 回復薬×3 + 解毒剤×2 でOK
  - 「武器を3つ」→ 剣×2 + 杖×1 でOK
```

#### 属性条件

```
条件: 属性.指定属性 >= 指定値
判定: 完成品の属性値が条件を満たすか
例:
  - 「火属性10以上の武器」→ 火山石を使った剣など
  - 「水属性15以上の薬」→ 清水を多く使った回復薬
```

#### 効果ベース

```
条件: 効果.指定効果 >= 指定値
判定: 完成品の効果値が条件を満たすか
例:
  - 「HP50回復できるもの」→ 高品質な回復薬など
  - 「攻撃力+10の武器」→ 強化された鋼の剣
```

#### 素材消費

```
条件: 使用素材 IN 指定条件
判定: 調合時に使用した素材が条件を満たすか
例:
  - 「レア素材を3つ使ったアイテム」→ 高級素材×3以上使用
  - 「火山石を使った品」→ 火山石を素材に含む調合品
```

#### 複合条件

```
条件: 条件1 AND 条件2 [AND 条件3...]
判定: すべての条件を満たすか
例:
  - 「火属性10以上で品質B以上」→ 両方の条件を満たす品
  - 「薬カテゴリで数量3以上、品質C以上」→ 3つの条件すべて
