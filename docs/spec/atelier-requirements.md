## 概要（旧版・参考用）

本ゲームは、錬金術をテーマにした短期ローグライク × デッキ構築ゲームである。
プレイヤーは錬金術師として、カードを組み合わせて顧客の依頼を達成し、工房を発展させる。

**ジャンル**: 錬金術 × デッキ構築 × ローグライク
**コンセプト**: 「混ぜて、選んで、満足させろ。」
**プレイ時間**: 1プレイ 20〜40分

### ワンセンテンス・コンセプト

> 錬金術師（プレイヤー）が、カードを組み合わせて錬金を行い、
> 素材の在庫と依頼を見比べて何を作るか、どういう素材を集めるかを判断し、
> 「在庫がなくなるスリル」と「ギリギリで依頼を達成した時の快感」を味わうゲーム

### 中核の面白さ

このゲームは「在庫（カード）が減り続ける恐怖の中で、今使うか温存するかを選び続けることが楽しい」

## 1. 体験設計

### 1.1 プレイサイクル

1プレイ（20〜40分）の中で、以下のループを繰り返す。

#### メインループ（1ターン: 30秒〜1分）

1. **状況提示**: 手札配布（5枚まで）、受注中の依頼（最大3件）、デッキ残量表示
2. **プレイヤーの選択**: カード使用/捨てる/ターン終了
3. **結果反映**: 使用カードは完全消滅（★核心）、依頼進行度更新、報酬獲得
4. **フィードバック**: 効果音・演出、在庫警告（残り10枚以下）

### 1.2 中核判断

> **「このカードを今使うべきか？」**

- 判断場面: 毎ターン、手札プレイ時
- 選択肢: 今使って依頼達成 / 温存 / 捨てて別カードを引く
- 面白さ: **使ったカードは二度と戻らない（在庫が減る恐怖）**

### 1.3 勝敗条件

| 条件 | 内容 |
|------|------|
| **勝利** | 最終依頼をクリア |
| **敗北** | デッキ0枚 = **即ゲームオーバー** |

### 1.4 プレイヤー操作

- 選ぶ、使う、捨てる、温存する、確認する
- 購入する、強化する、受注する

## 2. ゲーム要素

### 2.1 プレイヤー
| 属性 | 内容 |
|------|------|
| デッキ（在庫） | カード使用で減少、報酬で増加 |
| 手札 | 毎ターン5枚まで補充 |
| エナジー | 毎ターン3回復、最大10 |

### 2.2 カード
素材/操作/知識/特殊/アーティファクトの5系統（触媒は廃止）

### 2.3 依頼
同時3件、難易度1-5星、報酬=新カード+通貨+名声

---

## 3. 数値設計

### 3.1 在庫管理（★核心）
| 変数 | 値 |
|------|-----|
| カード使用時 | **完全消滅** |
| 在庫警告 | 残り10枚以下 |
| ゲームオーバー | 残り0枚 |

### 3.2 エナジー
ターン開始: 3、最大: 10、カードコスト: 0-5

### 3.3 手札
初期: 5枚、毎ターン: 5枚まで補充
