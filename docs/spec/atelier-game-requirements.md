## 概要

本ドキュメントは、錬金術をテーマにした開拓系デッキ構築ローグライクゲームの要件定義書である。
game-requirementsスキルのワークフローに沿って、プレイサイクル・勝敗条件・操作・要素・数値・プレイフローを定義する。

**ジャンル**: 錬金術 × デッキ構築 × ローグライク × 開拓
**コンセプト**: 「受けて、採って、混ぜて、届けろ。」
**プレイ時間**: 1プレイ 30〜45分（3開拓地）

### ワンセンテンス・コンセプト

> 錬金術師（プレイヤー）が、依頼受注・採取・調合・依頼納品・買い物を自由に選び、
> 期限と状況を見て何を優先すべきかを判断しながら、3つの開拓地を攻略する
> 「リソース管理のスリル」と「計画達成の快感」を味わうゲーム

### 中核の面白さ

> このゲームは「消耗度が上がり続け、リソースと依頼期限が限られている中で、依頼受注・採取・調合・依頼納品・買い物のどれを今やるべきか、自分の状況を見て判断し続けることが楽しい」

## 1. プレイサイクル

### 1.1 全体構造（3開拓地制）

```
開拓地1（10〜15分）→ 開拓地2（10〜15分）→ 開拓地3（10〜15分）→ ゲームクリア
     ↓                    ↓                    ↓
  [継承: デッキ+通貨+アーティファクト]
```

- **合計プレイ時間**: 30〜45分
- **開拓地ごとに難易度上昇**: 消耗度の上昇速度が増加

### 1.2 開拓地内のメインループ（1ターン: 10〜30秒）

```
① 状況把握: お金・素材・消耗度・開拓度・受注中の依頼と期限を確認
② プレイヤーの選択:
   - 依頼を受注する（期限と報酬を確認、計画を立てる）
   - 採取に行く（素材獲得、ターン消費）
   - 調合する（武器・薬を作成、素材消滅）
   - 依頼を納品する（武器→開拓度UP+お金、薬→消耗度DOWN+お金）
   - 買い物する（素材/アーティファクト獲得、お金消費）
③ 結果反映:
   - 選択した行動の効果が発動
   - 納品した武器・薬に応じて開拓度UP/消耗度DOWN + お金獲得
④ ターン経過: 消耗度が自然上昇、依頼期限が減少
⑤ イベント判定: 開拓度達成でアーティファクト獲得、ランダムイベント発生
→ ①に戻る（開拓度100% or 消耗度100%まで）
```

### 1.3 ターン構造

| 要素 | 詳細 |
|------|------|
| **開拓地全体のターン数** | 日数（30日）× 時間帯（朝・昼・夕=3）= 90ターン |
| **採取** | 1〜3ターン消費（採取地の距離による） |
| **調合** | 1〜3ターン消費（アイテムの複雑さによる） |
| **買い物** | 1ターン消費 |
| **依頼受注** | 0ターン消費（いつでも依頼リストを確認・受注可能） |
| **依頼納品** | 0ターン消費（調合済みの武器・薬があれば即納品可能） |
| **消耗度上昇** | 毎ターン自動上昇（開拓地により異なる） |

## 2. 勝敗条件

| 条件 | 内容 |
|------|------|
| **勝利条件** | 3つの開拓地すべてで開拓度100%達成 |
| **敗北条件** | 消耗度100% = **即ゲームオーバー** |
| **継続不能条件** | デッキ0枚かつ購入不可かつ採取不可（詰み状態） |

## 3. プレイヤー操作

動詞で列挙：

### 採取フェーズ（新規）
- **採取地を選ぶ**: 複数の採取地から行き先を選択
- **採取する**: 選んだ場所で素材を集める（ターン消費）
- **イベントに対応する**: 天候変化やレア素材発見に対応

### 調合フェーズ
- **選ぶ**: 作りたいアイテムカテゴリを選択（武器/薬/道具）
- **セットする**: デッキから素材カードをスロットにセット
- **調合する**: 属性値を確認してアイテムを完成させる
- **キャンセルする**: 調合をやめて素材を戻す

### 依頼フェーズ
- **依頼を見る**: 受注可能な依頼一覧を確認（期限・報酬を確認）
- **依頼を受ける**: 依頼を受注する（最初にやるべき行動）
- **計画を立てる**: 期限内に達成できるか計算する
- **納品する**: 調合済みの武器・薬を納品（効果発動+お金獲得）
  - 武器を納品 → 開拓度UP + お金獲得
  - 薬を納品 → 消耗度DOWN + お金獲得
- **依頼を破棄する**: 受注中の依頼を破棄（ペナルティあり）

### 買い物フェーズ
- **素材を購入する**: 通貨で素材カードを購入（ランダムラインナップ）
- **アーティファクトを購入する**: 通貨でアーティファクトを購入
- **更新する**: ショップのラインナップを更新（ターン消費）

### 確認系
- **デッキを見る**: 所持している全素材カードを確認
- **指標を確認する**: 開拓度・消耗度の現在値を確認
- **お金を確認する**: 所持金を確認 
- **アーティファクトを見る**: 獲得済みアーティファクトを確認
- **依頼状況を見る**: 受注中の依頼と進捗を確認

## 4. ゲーム要素

### 4.1 カード系統（5系統）

| 系統 | 役割 | 効果例 |
|------|------|--------|
| **素材** | 属性値を提供 | 火属性+3、水属性+2 |
| **操作** | 属性を変換/増幅 | 火→水変換、属性2倍 |
| **知識** | 追加効果 | 調合ターン-1、品質UP |
| **特殊** | 条件付き強力効果 | 特定カテゴリ調合時に効果3倍 |
| **アーティファクト** | 永続パッシブ | 消耗度上昇-10%、開拓度ボーナス+5% |

### 4.2 属性システム

4属性制（素材側で属性の偏りを表現）:

| 属性 | 特徴 | 傾向のある素材 |
|------|------|---------------|
| **火** | 攻撃的、高い属性値 | 鉱物系（鉄鉱石、硫黄） |
| **水** | 回復的、安定した効果 | ハーブ系（薬草、苔） |
| **土** | 持続的、防御効果 | 鉱物系（粘土、石） |
| **風** | 軽量、コスト軽減 | ハーブ系（花、種） |

**重要**: 素材には複数の属性がある（例: 鉄鉱石 = 火+3、土+2）

### 4.3 アイテムカテゴリ（3カテゴリ）

| カテゴリ | 納品時の効果 | 備考 |
|---------|-------------|------|
| **武器** | 開拓度UP（15〜25%）+ お金獲得 | 開拓の主力、**納品時に効果発動** |
| **薬** | 消耗度DOWN（15〜25%）+ お金獲得 | 消耗回復の主力、**納品時に効果発動** |
| **道具** | - | **アーティファクト化**（パッシブ効果、納品不可） |

**重要**: 武器・薬は**依頼納品が唯一の使用方法**。納品時に効果発動とお金獲得の両方を得られる。

### 4.4 指標システム（2指標）

| 指標 | 役割 | 増減方法 |
|------|------|---------|
| **開拓度** | 100%でクリア | 武器の納品で上昇 |
| **消耗度** | 100%で失敗 | ターン経過で自然上昇、薬の納品で減少 |

### 4.5 採取システム（新規）

#### 採取地の設計

| 採取地 | 距離 | ターン消費 | 取れる素材 | 特徴 |
|--------|------|----------|----------|------|
| **近郊の森** | 近い | 1ターン | 薬草、花、木の実 | 安全、水・風属性多め |
| **山麓の岩場** | 中距離 | 2ターン | 鉄鉱石、石、粘土 | 火・土属性多め |
| **奥地の洞窟** | 遠い | 3ターン | レア鉱石、珍しい苔 | レア素材多め、イベント発生率高 |
| **古代遺跡** | 遠い | 3ターン | 特殊素材、古代の欠片 | 高品質素材、危険なイベントも |

#### 採取イベント

| イベント | 発生確率 | 効果 |
|---------|---------|------|
| **晴天** | 40% | 通常通り採取 |
| **雨天** | 20% | 水属性素材+1、採取量やや減少 |
| **強風** | 15% | 風属性素材+1、採取量やや減少 |
| **レア素材発見** | 10% | 高品質な素材を追加で獲得 |
| **天候悪化** | 10% | 採取量半減、追加ターン消費の可能性 |
| **危険遭遇** | 5% | 消耗度上昇、素材獲得減少 |

### 4.6 依頼システム（受注→納品型）

#### 依頼の流れ

```
① 依頼受注: 依頼リストから選んで受注（期限・報酬を確認）
② 計画: 期限内に達成するために何ターン必要か考える
③ 素材調達: 採取または買い物で素材を集める
④ 調合: 依頼に必要な武器・薬を調合
⑤ 納品: 調合した武器・薬を納品（効果発動+お金獲得）
```

#### 依頼の構造

| 要素 | 詳細 |
|------|------|
| **依頼内容** | 武器または薬を納品 |
| **報酬** | 通貨（お金）+ アイテム効果（武器→開拓度UP、薬→消耗度DOWN） |
| **期限** | 受注から一定ターン以内（**必須**） |
| **難易度** | 必要な属性値の合計で決定 |

#### 依頼の例

| 依頼名 | 必要アイテム | 報酬 | 効果 | 期限 |
|--------|------------|------|------|------|
| 薬草の調合 | 薬（水属性10以上） | 50G | 消耗度-15% | 15ターン |
| 鋼の剣 | 武器（火属性15以上） | 100G | 開拓度+15% | 15ターン |
| 治癒の霊薬 | 薬（水属性20以上、品質B以上） | 200G | 消耗度-20% | 10ターン |
| 伝説の武具 | 武器（全属性10以上） | 500G | 開拓度+25% | 8ターン |

#### 同時受注

- **最大受注数**: 3件
- **依頼の更新**: 開拓度25%/50%/75%達成時に新しい依頼が追加
- **期限切れ**: 期限内に納品できないと依頼失敗（ペナルティなし、次の依頼を受注可能）

### 4.7 買い物システム（改良版）

#### 素材ショップ

| 要素 | 詳細 |
|------|------|
| **ラインナップ** | ランダムで3〜5種類の素材が並ぶ |
| **価格** | 素材のレアリティで変動（10G〜100G） |
| **更新** | 1ターン消費で更新可能、または自動更新（一定ターンごと） |

#### アーティファクトショップ

| 要素 | 詳細 |
|------|------|
| **ラインナップ** | 2〜3種類のアーティファクトが並ぶ |
| **価格** | 高価（200G〜500G） |
| **効果** | 強力なパッシブ効果 |

### 4.8 アーティファクト

#### 獲得方法

| 方法 | 詳細 |
|------|------|
| **道具を調合** | 道具カテゴリで調合するとアーティファクト化 |
| **開拓度達成** | 25%/50%/75%達成時に3択から選択（無料） |
| **ショップで購入** | お金で購入（高価） |
| **ランダムイベント** | ボーナスイベントで獲得 |

#### アーティファクト例

| 名前 | 効果 | 入手方法 |
|------|------|---------|
| 頑丈な靴 | 消耗度上昇-10%/ターン | 調合/選択 |
| 開拓者の斧 | 開拓度ボーナス+5% | 調合/選択 |
| 商人の護符 | 買い物10%OFF | 購入/選択 |
| 錬金術師の手袋 | 調合ターン消費-1 | 調合/選択 |
| 幸運のお守り | レア素材出現率2倍 | 購入/選択 |
| 採取者のカバン | 採取時の素材+1 | 調合/選択 |
| 依頼の紋章 | 依頼報酬+20% | 購入/選択 |

## 5. ランダム性（ローグライク要素）

### 5.1 採取のランダム化

| 要素 | 設計 |
|------|------|
| **天候** | 採取地ごとにランダムで天候が変化 |
| **獲得素材** | 採取地の傾向に基づきランダム決定 |
| **レア素材** | 低確率で高品質素材が出現（5〜10%） |
| **イベント** | 採取中にランダムイベント発生 |

### 5.2 ショップのランダム化

| 要素 | 設計 |
|------|------|
| **素材ラインナップ** | 買い物のたびにランダム更新 |
| **価格変動** | 素材レアリティで変動 |
| **アーティファクト** | 開拓地ごとに異なるラインナップ |

### 5.3 依頼のランダム化

| 要素 | 設計 |
|------|------|
| **依頼内容** | 開拓地ごとにランダム生成 |
| **報酬** | 難易度に応じて変動 |
| **追加依頼** | 開拓度達成時に新しい依頼が追加 |

### 5.4 開拓度達成でアーティファクト獲得

| 開拓度 | イベント | 選択肢 |
|--------|---------|--------|
| **25%** | アーティファクト選択 | 3つから1つ |
| **50%** | アーティファクト選択 | 3つから1つ |
| **75%** | **レアアーティファクト**選択 | 2つから1つ |

※各開拓地でそれぞれ3回の獲得機会 → 1プレイで最大9個獲得

## 7. 想定プレイフロー

### 開拓地1（序盤）

#### 1ターン目
- **状況**: デッキ10枚、お金100G、開拓度0%、消耗度0%
- **行動**: **依頼受注**（武器の依頼: 火属性15以上、報酬100G、開拓度+15%、期限15ターン）
- **判断**: 「15ターンで達成できるか？採取2回+調合1回で約5ターン。余裕がある」
- **結果**: 依頼を受注

#### 2ターン目
- **状況**: デッキ10枚、お金100G、開拓度0%、消耗度1%、依頼期限残り14ターン
- **行動**: 採取（山麓の岩場）→ 火・土属性の素材3枚獲得
- **結果**: デッキ13枚、ターン消費2、消耗度+2%

#### 4ターン目
- **状況**: デッキ13枚、お金100G、開拓度0%、消耗度4%、依頼期限残り11ターン
- **行動**: 武器を調合（火属性素材3枚消費）
- **結果**: デッキ10枚、武器完成

#### 5ターン目
- **状況**: デッキ10枚、お金100G、開拓度0%、消耗度5%、依頼期限残り10ターン、武器1個
- **行動**: **依頼納品**（武器を納品）
- **結果**: 開拓度+15%、お金+100G = 200G（効果とお金両方獲得！）

#### 15ターン目
- **状況**: デッキ7枚、お金180G、開拓度25%、消耗度15%
- **イベント**: 開拓度25%達成 → アーティファクト選択
- **選択**: 「採取者のカバン」（採取時の素材+1）を獲得

#### 25ターン目
- **状況**: デッキ10枚、お金150G、開拓度50%、消耗度30%
- **判断**: 消耗度が上がってきた。薬の依頼を受注して消耗度を下げつつお金も稼ごう
- **行動**: **依頼受注**（薬の依頼: 水属性10以上、報酬50G、消耗度-15%、期限15ターン）
- **結果**: 薬の依頼を受注、採取→調合→納品で消耗度を管理

#### 35ターン目
- **状況**: デッキ8枚、お金200G、開拓度75%、消耗度25%
- **行動**: 薬を納品済み（消耗度-15%）、さらに武器の依頼を受注・納品
- **結果**: 消耗度を抑えながら開拓度を伸ばす

#### 40ターン目（開拓地1クリア）
- **状況**: デッキ8枚、お金250G、開拓度100%、消耗度40%
- **結果**: 開拓地1クリア！デッキ・お金・アーティファクトを持って開拓地2へ

## 8. 拡張要素（将来実装）

### 8.1 職業システム

| 職業 | 得意属性 | 得意行動 | 特徴 |
|------|---------|---------|------|
| 錬金術師 | なし | 調合 | 基本職業（バランス型） |
| 鍛冶屋 | 火・土 | 武器調合 | 武器の品質ボーナス |
| 薬師 | 水・風 | 薬調合 | 薬の効果UP |
| 採取師 | なし | 採取 | 採取効率UP、イベント有利 |
| 商人 | なし | 依頼・買い物 | 報酬UP、価格DOWN |

- **解放条件**: 錬金術師でゲームクリア
- **Slay the Spire式**: 1ランで1職業を使用

### 8.2 追加アイテムカテゴリ

今後拡張予定:
- **食料**: 消耗度回復（小）
- **建築**: 開拓度上昇（中）+ 消耗度回復（小）
- **装飾品**: 通貨ボーナス、特殊効果

### 8.3 採取地タイプの拡張

将来の拡張で検討:
- **砂漠**: 火属性素材多め、消耗度上昇+0.5%/採取
- **氷原**: 水属性素材多め、ターン消費+1
- **火山**: 最高品質の火属性素材、危険イベント多め
- **精霊の森**: 全属性バランス良く、レア率高め
