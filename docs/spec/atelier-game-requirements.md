# アトリエ錬金術ゲーム 要件定義書

**バージョン**: 4.0
**作成日**: 2025-12-24
**更新日**: 2025-12-25
**形式**: game-requirementsスキル準拠

---

## 概要

本ドキュメントは、錬金術をテーマにした開拓系デッキ構築ローグライクゲームの要件定義書である。
game-requirementsスキルのワークフローに沿って、プレイサイクル・勝敗条件・操作・要素・数値・プレイフローを定義する。

**ジャンル**: 錬金術 × デッキ構築 × ローグライク × 開拓
**コンセプト**: 「混ぜて、選んで、開拓しろ。」
**プレイ時間**: 1プレイ 30〜45分（3開拓地）

### ワンセンテンス・コンセプト

> 錬金術師（プレイヤー）が、素材カードを自由に調合してアイテムを作り、
> 開拓度を上げつつ消耗度を抑えながら、3つの開拓地を攻略する
> 「リソース管理のスリル」と「調合の自由度」を味わうゲーム

### 中核の面白さ

> このゲームは「素材カードが減り続ける中で、武器（開拓）と薬（回復）のバランスを取りながら調合を選び続けることが楽しい」

---

## 1. プレイサイクル

### 1.1 全体構造（3開拓地制）

```
開拓地1（10〜15分）→ 開拓地2（10〜15分）→ 開拓地3（10〜15分）→ ゲームクリア
     ↓                    ↓                    ↓
  [継承: デッキ+通貨+アーティファクト]
```

- **合計プレイ時間**: 30〜45分
- **開拓地ごとに難易度上昇**: 消耗度の上昇速度が増加

### 1.2 開拓地内のメインループ（1ターン: 10〜30秒）

```
① 状況提示: デッキ全体表示、開拓度・消耗度表示、ターン数表示
② プレイヤーの選択: 調合する/買い物する/何もしない
③ 結果反映:
   - 調合 → 使用カードは完全消滅（★核心）、アイテム効果発動
   - 買い物 → 素材カード獲得
④ ターン経過: 消耗度が自然上昇
⑤ イベント判定: 開拓度達成でアーティファクト獲得、ランダムイベント発生
→ ①に戻る（開拓度100% or 消耗度100%まで）
```

### 1.3 ターン構造

| 要素 | 詳細 |
|------|------|
| **開拓地全体のターン数** | 日数（30日）× 時間帯（朝・昼・夕=3）= 90ターン |
| **調合** | 1〜2ターン消費（アイテムの複雑さによる） |
| **買い物** | 1ターン消費 |
| **消耗度上昇** | 毎ターン自動上昇（開拓地により異なる） |

---

## 2. 勝敗条件

| 条件 | 内容 |
|------|------|
| **勝利条件** | 3つの開拓地すべてで開拓度100%達成 |
| **敗北条件** | 消耗度100% = **即ゲームオーバー** |
| **継続不能条件** | デッキ0枚かつ購入不可（詰み状態） |

---

## 3. プレイヤー操作

動詞で列挙：

### 調合フェーズ
- **選ぶ**: 作りたいアイテムカテゴリを選択（武器/薬/道具）
- **セットする**: デッキから素材カードをスロットにセット
- **調合する**: 属性値を確認してアイテムを完成させる
- **キャンセルする**: 調合をやめて素材を戻す

### 買い物フェーズ
- **購入する**: 通貨で素材カードを購入
- **更新する**: ショップのラインナップを更新（ターン消費）

### 確認系
- **デッキを見る**: 所持している全素材カードを確認
- **指標を確認する**: 開拓度・消耗度の現在値を確認
- **アーティファクトを見る**: 獲得済みアーティファクトを確認

## 4. ゲーム要素

### 4.1 カード系統（5系統）

| 系統 | 役割 | 効果例 |
|------|------|--------|
| **素材** | 属性値を提供 | 火属性+3、水属性+2 |
| **操作** | 属性を変換/増幅 | 火→水変換、属性2倍 |
| **知識** | 追加効果 | 調合ターン-1、品質UP |
| **特殊** | 条件付き強力効果 | 特定カテゴリ調合時に効果3倍 |
| **アーティファクト** | 永続パッシブ | 消耗度上昇-10%、開拓度ボーナス+5% |

### 4.2 属性システム

4属性制（素材側で属性の偏りを表現）:

| 属性 | 特徴 | 傾向のある素材 |
|------|------|---------------|
| **火** | 攻撃的、高い属性値 | 鉱物系（鉄鉱石、硫黄） |
| **水** | 回復的、安定した効果 | ハーブ系（薬草、苔） |
| **土** | 持続的、防御効果 | 鉱物系（粘土、石） |
| **風** | 軽量、コスト軽減 | ハーブ系（花、種） |

**重要**: 素材には複数の属性がある（例: 鉄鉱石 = 火+3、土+2）

### 4.3 アイテムカテゴリ（3カテゴリ）

| カテゴリ | 開拓度への影響 | 消耗度への影響 | 備考 |
|---------|--------------|--------------|------|
| **武器** | +上昇（15〜25%） | - | 開拓の主力 |
| **薬** | - | -減少（15〜25%） | 消耗回復の主力 |
| **道具** | - | - | **アーティファクト化**（パッシブ効果） |

**設計方針**: 初期は3カテゴリでシンプルに。今後拡張予定（食料、建築、装飾品など）

### 4.4 指標システム（2指標）

| 指標 | 役割 | 増減方法 |
|------|------|---------|
| **開拓度** | 100%でクリア | 武器で上昇 |
| **消耗度** | 100%で失敗 | ターン経過で自然上昇、薬で減少 |

### 4.5 アーティファクト

#### 獲得方法

| 方法 | 詳細 |
|------|------|
| **道具を調合** | 道具カテゴリで調合するとアーティファクト化 |
| **開拓度達成** | 25%/50%/75%達成時に3択から選択（コストなし） |
| **ランダムイベント** | ボーナスイベントで獲得 |

#### アーティファクト例

| 名前 | 効果 |
|------|------|
| 頑丈な靴 | 消耗度上昇-10%/ターン |
| 開拓者の斧 | 開拓度ボーナス+5% |
| 商人の護符 | 買い物10%OFF |
| 錬金術師の手袋 | 調合ターン消費-1 |
| 幸運のお守り | レア素材出現率2倍 |

## 5. ランダム性（ローグライク要素）

### 5.1 ショップのランダム化

| 要素 | 設計 |
|------|------|
| **商品ラインナップ** | 買い物のたびにランダム更新 |
| **価格変動** | 素材レアリティで変動 |
| **レア素材** | 低確率で高品質素材が出現（5〜10%） |
| **ラインナップ数** | 3〜5種類の素材から選択 |

### 5.2 ランダムイベントシステム

| タイミング | 発生確率 | 例 |
|-----------|---------|-----|
| **ターン開始時** | 10〜15% | ランダムイベント発生 |
| **開拓地到着時** | 100% | 開拓地固有イベント（初回のみ） |
| **特定条件達成時** | 100% | 消耗度50%到達時など |

**イベントの種類**:

| タイプ | 内容例 |
|-------|-------|
| **選択イベント** | 「素材+3枚 or 消耗度-10%」どちらかを選ぶ |
| **ボーナスイベント** | 無条件で素材やアーティファクトを獲得 |
| **ピンチイベント** | 消耗度急上昇、素材1枚損失 |
| **ショップイベント** | 特別な商品（レア素材、割引）が出現 |
| **調合イベント** | 次の調合で属性値ボーナス |

### 5.3 開拓度達成でアーティファクト獲得

| 開拓度 | イベント | 選択肢 |
|--------|---------|--------|
| **25%** | アーティファクト選択 | 3つから1つ |
| **50%** | アーティファクト選択 | 3つから1つ |
| **75%** | **レアアーティファクト**選択 | 2つから1つ |

※各開拓地でそれぞれ3回の獲得機会 → 1プレイで最大9個獲得

## 6. 数値設計

### 6.1 開拓地パラメータ

| パラメータ | 開拓地1 | 開拓地2 | 開拓地3 |
|-----------|--------|--------|--------|
| 初期ターン数 | 90 | 90 | 90 |
| 開拓度目標 | 100% | 100% | 100% |
| 消耗度上昇/ターン | +1% | +1.5% | +2% |
| 購入可能素材の質 | 低 | 中 | 高 |

### 6.2 素材カードのバランス

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| 初期デッキ | 10枚 | 緊張感高め |
| デッキ上限 | 20枚 | アーティファクトで拡張可能 |
| 1調合の素材消費 | 2〜5枚 | アイテムの複雑さによる |
| 買い物の素材獲得 | 1〜3枚 | 通貨と引き換え |

### 6.3 アイテムの効果量

| カテゴリ | 開拓度影響 | 消耗度影響 | 必要属性値 | 備考 |
|---------|-----------|-----------|----------|------|
| 武器 | +15〜25% | - | 合計15以上 | 開拓の主力 |
| 薬 | - | -15〜25% | 合計15以上 | 消耗回復の主力 |
| 道具 | - | - | 合計10以上 | アーティファクト化（パッシブ効果） |

### 6.4 調合システム

- **手札なし**: デッキ（所持している素材カード）全体にアクセス可能
- **調合時**: 使用した素材カードは**完全消滅**
- **ターン消費**: 簡単なアイテム=1ターン、複雑なアイテム=2ターン
- **調合上限**: なし（ターンが続く限り何度でも調合可能）

**調合の流れ**:
```
① 作りたいアイテムカテゴリを選択（武器/薬/道具）
② 素材カードを選択してスロットにセット
③ 属性値の合計が一定以上でアイテム完成
④ 完成したアイテムのカテゴリと品質に応じて効果発動
   - 武器: 開拓度UP
   - 薬: 消耗度DOWN
   - 道具: アーティファクトとして永続パッシブ効果を獲得
⑤ 使用した素材カードは消滅
```

### 6.5 在庫管理の緊張感設計（★核心メカニクス）

| デッキ残量 | 状態 | UI表示 |
|-----------|------|--------|
| 15枚以上 | 安全 | 緑 |
| 10〜14枚 | 注意 | 黄色 |
| 5〜9枚 | 危険 | オレンジ |
| 1〜4枚 | 緊急 | 赤点滅 |
| 0枚 | 継続不能（購入不可なら敗北） | - |

## 7. 想定プレイフロー

### 開拓地1（序盤）

#### 1ターン目
- **状況**: デッキ10枚、開拓度0%、消耗度0%
- **行動**: 買い物 → 素材2枚獲得
- **結果**: デッキ12枚、ターン消費、消耗度+1%

#### 5ターン目
- **状況**: デッキ12枚、開拓度0%、消耗度5%
- **行動**: 武器を調合（素材3枚消費）
- **結果**: デッキ9枚、開拓度+20%、消耗度+1%

#### 15ターン目
- **状況**: デッキ8枚、開拓度40%、消耗度25%
- **判断**: 消耗度が上がってきた。薬を作るか、武器で一気に開拓するか
- **行動**: 薬を調合（素材2枚消費）
- **結果**: デッキ6枚、消耗度-20% → 6%

#### 25ターン目（アーティファクト獲得）
- **状況**: デッキ10枚、開拓度50%、消耗度30%
- **イベント**: 開拓度50%達成 → アーティファクト選択
- **選択**: 「頑丈な靴」（消耗度上昇-10%/ターン）を獲得

#### 40ターン目（開拓地1クリア）
- **状況**: デッキ7枚、開拓度100%、消耗度45%
- **結果**: 開拓地1クリア！デッキ・通貨・アーティファクトを持って開拓地2へ

### 開拓地2（中盤）

#### 41ターン目
- **状況**: デッキ7枚、開拓度0%、消耗度0%（リセット）
- **注意**: 消耗度上昇が+1.5%/ターンに増加
- **行動**: 買い物でデッキ補充

### 開拓地3（終盤）

#### 80ターン目
- **状況**: デッキ5枚、開拓度75%、消耗度70%
- **緊張**: 消耗度上昇+2%/ターン、残りターンでクリアできるか
- **行動**: 残り素材で武器を連続調合
- **結果**: 開拓度100%達成！ゲームクリア
