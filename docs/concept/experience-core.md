## 1. 中核判断

### プレイヤーの唯一の問い

```
プレイヤーは常に「今の状況で、何を優先すべきか？」を考えている
```

### 判断の詳細

| 項目 | 内容 |
|------|------|
| 判断が発生する場面 | 毎ターン、次の行動を決める時 |
| 選択肢の例 | 依頼受注 / 採取（素材獲得）/ 調合（武器・薬）/ 依頼納品（効果発動+お金獲得）/ 買い物（素材・アーティファクト） |
| 判断を面白くする要素 | 状況（お金・素材・消耗度・開拓度・依頼期限）によって最適解が変わる |

### 状況に応じた判断の例

- **最初にやること** → 依頼を受注し、期限内に達成できるか計画を立てる
- **お金が少ない** → 依頼を受注して武器・薬を納品し、報酬を得る
- **素材が足りない** → 採取に行く（近い/遠い、リスク/報酬）
- **消耗度が高い** → 薬の依頼を受注して納品（消耗度DOWN + お金獲得）
- **開拓度を上げたい** → 武器の依頼を受注して納品（開拓度UP + お金獲得）
- **アーティファクトが欲しい** → 依頼納品でお金を貯めて買い物

## 3. 判断→結果→学習ループ

### メインループ

```
状況把握: お金・素材・消耗度・開拓度・受注中の依頼と期限を確認

判断: 今何をすべきか？
  1. 依頼受注 → どの依頼を受けるか選ぶ（期限・報酬を考慮）
  2. 採取 → 素材獲得（時間消費、場所によりリターンが異なる）
  3. 調合 → 武器・薬を作成（素材消滅）
  4. 依頼納品 → 武器・薬を納品
     - 武器納品 → 開拓度UP + お金獲得
     - 薬納品 → 消耗度DOWN + お金獲得
  5. 買い物 → 素材/アーティファクト獲得（お金消費）

結果:
  - リソースの増減
  - 時間（ターン）の消費
  - 消耗度の自然上昇
  - 依頼期限の減少
  - ランダムイベントの発生

学習: 状況に応じた優先順位の感覚を習得
  - どの依頼を受注すべきか
  - 期限内に達成するために何ターン必要か
```

### ループ詳細

| 項目 | 内容 |
|------|------|
| ループの回転速度 | 1ターン（10〜30秒） |
| 結果が判明するまでの時間 | 即座（調合効果）〜数ターン後（リソース枯渇） |
| 学習の蓄積方法 | プレイヤースキル（状況判断力） + ゲーム内成長（アーティファクト） |

## 4. リソース設計

### プライマリリソース

| リソース名 | 役割 | 獲得方法 | 消費タイミング |
|-----------|------|---------|---------------|
| 素材カード | 調合の材料 | **採取（メイン）**・買い物（サブ） | 調合時に完全消滅 |
| 通貨（お金） | 買い物の原資 | **依頼納品（唯一）** | 買い物で消費 |
| ターン | 行動制限 | 開拓地開始時（90ターン） | 採取・調合・買い物で消費 |

### 指標リソース（2指標）

| 指標名 | 役割 | 増減方法 |
|--------|------|---------|
| **開拓度** | 100%でクリア | 武器納品で上昇 |
| **消耗度** | 100%で敗北 | ターン経過で上昇、薬納品で減少 |

### リソース枯渇時の緊張感

| 状況 | 何が起きるか | プレイヤーの感情 | リカバリー手段 |
|------|------------|-----------------|---------------|
| 素材不足 | 調合したいのに調合できない | 焦り、計画の見直し | 採取に行く（時間消費） |
| お金不足 | 買い物できない、アーティファクトが買えない | 依頼の重要性を実感 | 依頼を達成する |
| ターン不足 | 開拓度100%に届かない | 効率化の必要性を実感 | 次のプレイで改善 |
| 消耗度上昇 | ゲームオーバーが近づく | 緊張感、判断の重み | 薬の依頼を受注して納品 |

## 7. 補足事項

### 設計上の注意点

- **状況判断**が核心 → 同じ行動でも状況によって正解が変わる
- **依頼受注が最初** → 期限を考慮して計画を立てる
- **採取がメイン**の素材獲得 → 買い物は緊急時とアーティファクト用
- **依頼納品が唯一**のアイテム使用方法 → 効果発動とお金獲得の両方を得る
- **5つの行動**のバランス → どれか1つに偏るとクリアできない

### 今後の検討事項

- 採取地の数と特徴のバランス
- 依頼の難易度・期限・報酬のバランス
- 依頼期限の適切な設定
- 天候イベントの種類と影響
- アーティファクトの価格設定

### 関連する他の体験要素

- 採取地のランダム化（天候、レア素材発見）
- ショップのランダム化（素材ラインナップ）
- 依頼のランダム化（種類、期限、報酬）
- 開拓度達成でのアーティファクト選択（3択から1つ）
- 職業システム（クリア後解放、将来の拡張）
