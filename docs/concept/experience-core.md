## 1. 中核判断

### プレイヤーの唯一の問い

```
プレイヤーは常に「ランクを倒すために、今何をすべきか？」を考えている
```

### 判断の詳細

| 項目 | 内容 |
|------|------|
| 判断が発生する場面 | 毎ターン、手札と素材を見て次の行動を決める時 |
| 選択肢の例 | 採取地カード使用 / レシピカード使用 / 依頼受注 / 依頼納品 / 買い物 / 休憩 |
| 判断を面白くする要素 | 手札の内容・素材状況・依頼期限・ランクHPによって最適解が変わる |

### 状況に応じた判断の例

- **ゲーム開始時** → 依頼を受注し、目標を設定する
- **素材が足りない** → 手札の採取地カードを使う
- **調合可能** → 手札のレシピカードで調合
- **完成品がある** → 依頼納品で貢献度獲得 → ランクにダメージ！
- **手札が偏っている** → 休憩アクションで手札入れ替え
- **デッキが弱い** → 買い物で新カード獲得
- **ランクHP残りわずか** → 高効率な依頼を優先してトドメ

## 3. 判断→結果→学習ループ

### メインループ

```
状況把握: ランクHP・手札・素材インベントリ・消耗度・受注依頼を確認

判断: ランクを倒すために今何をすべきか？
  1. 採取地カード使用 → 素材獲得（コスト1〜3）
  2. レシピカード使用 → 素材消費して調合（コスト1〜2）
  3. 依頼受注 → 目標設定、期限確認（コスト0）
  4. 依頼納品 → 貢献度獲得 → ランクにダメージ！（コスト0）
  5. 買い物 → 新カード/素材獲得（コスト1 + お金）
  6. 休憩 → 手札2枚捨てて2枚ドロー（コスト0）

結果:
  - リソースの増減
  - 貢献度の獲得 → ランクHPにダメージ
  - 消耗度の変動
  - 依頼期限の減少
  - ランクHP0 → 昇格試験へ！

学習: デッキ構築と行動優先順位の感覚を習得
  - どの採取地カードを入れるべきか
  - どのレシピカードが有効か
  - 期限内に達成できる依頼はどれか
```

### ループ詳細

| 項目 | 内容 |
|------|------|
| ループの回転速度 | 1日（30秒〜1分） |
| 結果が判明するまでの時間 | 即座（納品）〜数日後（依頼達成、ランクHP0） |
| 学習の蓄積方法 | プレイヤースキル（デッキ構築力、行動判断力） + ゲーム内成長（新カード獲得） |

## 4. リソース設計

### プライマリリソース

| リソース名 | 役割 | 獲得方法 | 消費タイミング |
|-----------|------|---------|---------------|
| 採取地カード | 素材獲得手段 | 初期デッキ・買い物・ランクアップ報酬 | 使用後デッキに戻る |
| レシピカード | 調合手段 | 初期デッキ・買い物・ランクアップ報酬 | 使用後デッキに戻る |
| 素材 | 調合の材料 | 採取地カード使用・買い物 | 調合時に**永続消費** |
| 通貨（お金） | 買い物の原資 | 依頼納品報酬 | 買い物で消費 |
| 行動ポイント | 行動制限 | 毎日3ポイント回復 | 各行動で消費 |

### 指標リソース

| 指標名 | 役割 | 増減方法 |
|--------|------|---------|
| **ランクHP** | 0でランクアップ | 依頼納品（貢献度）で減少 |
| **消耗度** | 100%で敗北/降格 | 依頼失敗で上昇、特定依頼で減少 |

### リソース枯渇時の緊張感

| 状況 | 何が起きるか | プレイヤーの感情 | リカバリー手段 |
|------|------------|-----------------|---------------|
| 素材不足 | 調合できない | 採取地カードの重要性を実感 | 採取地カードを使う |
| 採取地カード不足 | 素材が集められない | デッキ構築の重要性を実感 | 買い物で追加、休憩で手札入れ替え |
| レシピカード不足 | 調合できない | デッキバランスの重要性を実感 | 買い物で追加 |
| お金不足 | 新カードが買えない | 依頼納品の重要性を実感 | 依頼を達成する |
| 消耗度上昇 | 敗北/降格が近づく | 緊張感、慎重な判断 | 消耗度DOWN依頼を達成 |

## 7. 補足事項

### 設計上の注意点

- **敵＝ギルドランク** → 明確な目標と達成感
- **攻撃＝依頼納品** → 貢献度でランクにダメージ
- **デッキ＝能力** → 採取地＝行ける場所、レシピ＝作れるもの
- **素材＝弾薬** → 永続消費でリソース管理の重み
- **手札管理** → 採取地/レシピのバランスが戦略の鍵

### 今後の検討事項

- 各ランクのHP調整
- 昇格試験の難易度設計
- 採取地カードの種類と獲得素材バランス
- レシピカードの種類と必要素材バランス
- 依頼の報酬（貢献度）設計
- 消耗度の上昇/回復バランス

### 関連する他の体験要素

- 採取地カードごとの獲得素材ランダム性
- ショップのランダム化（カード・素材ラインナップ）

## 8. v4.0 主要変更点

| 要素 | v3.1 | v4.0 |
|------|------|------|
| 目標 | 開拓度100%達成 | **ギルドランクHP削り→昇格試験→ランクアップ** |

- **戦闘感の追加**: 依頼達成を「攻撃」として明確化

- **デッキの意味強化**: 採取地＝行ける場所、レシピ＝作れるもの
- **素材の重み増加**: 永続消費でリソース管理がシビアに
