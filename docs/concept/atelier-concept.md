## 1. ワンセンテンス・コンセプト

> 【誰が】【何をして】【どんな判断を繰り返し】【どんな感情になるゲームか】

```
錬金術師（プレイヤー）が
素材カードを自由に調合してアイテムを作り
開拓度を上げつつ消耗度を抑える調合バランスを判断し
「リソース管理のスリル」と「開拓完了の達成感」を味わうゲーム
```

**短縮版（30秒説明）:**
> 素材カードを調合して武器（開拓）と薬（回復）を作り、3つの開拓地を攻略するデッキ構築ローグライク

**キャッチコピー:**
> 「混ぜて、選んで、開拓しろ。」

## 4. 体験の強弱カーブ

### 1プレイ全体（30〜45分）

| 時間 | フェーズ | 判断内容 | 判断の重さ |
|------|---------|---------|-----------|
| 00:00-02:00 | 職業選択 | 初期デッキを選ぶ | 軽い |
| 02:00-15:00 | 開拓地1 | 基本の調合バランスを学ぶ | 軽〜中 |
| 15:00-30:00 | 開拓地2 | 消耗度上昇+1.5%/ターンに適応 | 中 |
| 30:00-45:00 | 開拓地3 | 消耗度+2%/ターン、ギリギリの判断 | 重い〜最大 |

### 1開拓地内（10〜15分）

| 開拓度 | フェーズ | 判断内容 | 緊張感 |
|--------|---------|---------|-------|
| 0〜25% | 序盤 | デッキ補充、武器調合開始 | 低 |
| 25〜50% | 前半 | アーティファクト獲得、バランス調整 | 中 |
| 50〜75% | 後半 | 消耗度管理、薬の調合判断 | 高 |
| 75〜100% | 終盤 | 残りリソースで開拓完了を目指す | 最大 |

### 緊張感カーブ

- **開拓地1**: 消耗度上昇が緩やか（+1%/ターン）、余裕を持って学習
- **開拓地2**: 消耗度上昇が加速（+1.5%/ターン）、バランス感覚が求められる
- **開拓地3**: 消耗度上昇が激しい（+2%/ターン）、ギリギリの判断の連続

## 6. v4.0 主要変更点

| 要素 | 旧版（v1.x） | 新版（v2.0） |
|------|-------------|-------------|
| ゲーム構造 | ノード進行（ルート選択） | 3開拓地制（各10〜15分） |
| 勝利条件 | 最終依頼クリア | 3開拓地すべて開拓度100% |
| 敗北条件 | デッキ0枚 | 消耗度100% |
| 調合システム | 手札5枚から選択 | デッキ全体にアクセス |
| アイテムカテゴリ | 依頼の属性条件 | 武器・薬・道具の3つ |
| 報酬システム | 依頼達成でカード選択 | 開拓度達成でアーティファクト選択 |
| ランダム性 | ノード配置 | ショップ・イベント・アーティファクト選択 |
