## 1. ワンセンテンス・コンセプト

> 【誰が】【何をして】【どんな判断を繰り返し】【どんな感情になるゲームか】

```
錬金術師（プレイヤー）が
依頼受注・採取・調合・依頼納品・買い物を自由に選び
期限と状況を見て何を優先すべきかを判断し
「リソース管理のスリル」と「計画達成の快感」を味わうゲーム
```

**短縮版（30秒説明）:**
> 依頼を受注し、素材を採取し、武器・薬を調合して納品。期限内に達成してお金と効果を獲得しながら、3つの開拓地を攻略するデッキ構築ローグライク

**キャッチコピー:**
> 「受けて、採って、混ぜて、届けろ。」

## 4. 体験の強弱カーブ

### 1プレイ全体（30〜45分）

| 時間 | フェーズ | 判断内容 | 判断の重さ |
|------|---------|---------|-----------|
| 00:00-02:00 | 職業選択 | 初期デッキを選ぶ | 軽い |
| 02:00-15:00 | 開拓地1 | 採取・調合・依頼のバランスを学ぶ | 軽〜中 |
| 15:00-30:00 | 開拓地2 | 消耗度上昇+1.5%/ターンに適応 | 中 |
| 30:00-45:00 | 開拓地3 | 消耗度+2%/ターン、ギリギリの判断 | 重い〜最大 |

### 1開拓地内（10〜15分）

| 開拓度 | フェーズ | 判断内容 | 緊張感 |
|--------|---------|---------|-------|
| 0〜25% | 序盤 | 採取で素材集め、依頼受注 | 低 |
| 25〜50% | 前半 | アーティファクト獲得、依頼達成でお金稼ぎ | 中 |
| 50〜75% | 後半 | 消耗度管理、薬の調合判断 | 高 |
| 75〜100% | 終盤 | 残りリソースで開拓完了を目指す | 最大 |

### 緊張感カーブ

- **開拓地1**: 消耗度上昇が緩やか（+1%/ターン）、余裕を持って学習
- **開拓地2**: 消耗度上昇が加速（+1.5%/ターン）、バランス感覚が求められる
- **開拓地3**: 消耗度上昇が激しい（+2%/ターン）、ギリギリの判断の連続

## 5. ゲームサイクル

### 新しいメインループ

```
┌──────────────────────────────────────────────────┐
│                    状況把握                       │
│  お金・素材・消耗度・開拓度・受注依頼と期限を確認 │
└──────────────────────────────────────────────────┘
                    ↓
    ┌────────────────────────────────────┐
    │  期限内に達成するには今何をすべきか？ │
    └────────────────────────────────────┘
      ↓         ↓         ↓         ↓         ↓
   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐
   │依頼 │  │採取 │  │調合 │  │依頼 │  │買物 │
   │受注 │  │    │  │    │  │納品 │  │    │
   └─────┘  └─────┘  └─────┘  └─────┘  └─────┘
      │         │         │         │         │
      ↓         ↓         ↓         ↓         ↓
   期限と    素材獲得   武器・薬   効果発動   素材/
   報酬確認  (ターン    作成      +お金獲得  アーティ
   計画立案   消費)    (素材消滅)  武器→開拓度UP  ファクト
                                 薬→消耗度DOWN
                    ↓
          ターン経過（消耗度上昇、依頼期限減少）
                    ↓
              次のターンへ
```

### 状況に応じた判断の例

| 状況 | 優先すべき行動 | 理由 |
|------|--------------|------|
| 最初にやること | 依頼受注 | 期限を確認し、達成計画を立てる |
| お金が少ない | 依頼納品（武器・薬） | お金がないとアーティファクトが買えない |
| 素材が足りない | 採取 | 素材がないと調合も依頼達成もできない |
| 消耗度が高い | 薬の依頼を受注→納品 | 消耗度DOWN + お金獲得 |
| 開拓度を上げたい | 武器の依頼を受注→納品 | 開拓度UP + お金獲得 |
| 強化したい | 買い物（アーティファクト） | パッシブ効果で有利に |
| 依頼期限が迫っている | 調合→納品を優先 | 期限切れを防ぐ |

## 7. v3.1 主要変更点

| 要素 | v3.0 | v3.1 |
|------|------|------|
| キャッチコピー | 「採って、混ぜて、稼いで、開拓しろ。」 | 「**受けて、採って、混ぜて、届けろ。**」 |
| 行動選択肢 | 採取/調合/依頼納品/買い物 | **依頼受注**/採取/調合/依頼納品/買い物 |
| 調合カテゴリ | 武器/薬/納品用 | **武器/薬のみ**（納品用は廃止） |
| 依頼 | 納品でお金獲得 | **受注→納品**（期限あり、効果発動+お金獲得） |
| 武器の効果 | 調合時に開拓度UP | **納品時に開拓度UP + お金獲得** |
| 薬の効果 | 調合時に消耗度DOWN | **納品時に消耗度DOWN + お金獲得** |

### v3.0 からの変更点（参考）

| 要素 | v2.0（開拓地システム） | v3.0（状況判断型） |
|------|----------------------|-------------------|
| キャッチコピー | 「混ぜて、選んで、開拓しろ。」 | 「採って、混ぜて、稼いで、開拓しろ。」 |
| 中核判断 | 「武器を作るか、薬を作るか？」 | 「今の状況で、何を優先すべきか？」 |
| 素材獲得 | 買い物のみ | **採取（メイン）**+ 買い物（サブ） |
| お金獲得 | 開拓地継承時 | **依頼納品（唯一）** |
| 面白さの源泉 | 武器vs薬のバランス | **状況に応じた行動選択** |
| アトリエ要素 | 調合のみ | **採取・依頼を復活** |
