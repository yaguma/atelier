## 1. ワンセンテンス・コンセプト

> 【誰が】【何をして】【どんな判断を繰り返し】【どんな感情になるゲームか】

```
錬金術師（プレイヤー）が
採取地カードとレシピカードのデッキを駆使し
依頼を達成して貢献度を稼ぎ、ギルドランクを上げていくことで
「戦略的デッキ構築」と「ランクアップの達成感」を味わうゲーム
```

**短縮版（30秒説明）:**
> 採取地カードで素材を集め、レシピカードで調合し、依頼を達成してギルドランクを上げていく錬金術師RPG。ランクを倒すたびに新しい採取地やレシピが解放される。

**キャッチコピー:**
> 「依頼を受けて、素材を採って、調合して、ランクを上げろ。」

## 4. 体験の強弱カーブ

### 1プレイ全体（60〜90分）

| ランク | 時間目安 | 判断内容 | 判断の重さ |
|--------|---------|---------|-----------|
| G | 00:00-10:00 | チュートリアル、基本を学ぶ | 軽い |
| F | 10:00-25:00 | 採取・調合のバランスを学ぶ | 軽〜中 |
| E | 25:00-40:00 | 依頼失敗ペナルティに適応 | 中 |
| D | 40:00-55:00 | 品質要求に対応 | 中〜重 |
| C-B | 55:00-75:00 | 期限短縮、品質要求上昇 | 重い |
| A-S | 75:00-90:00 | ギリギリの判断、最終試験 | 最大 |

### 1ランク内（10〜20分）

| HP残量 | フェーズ | 判断内容 | 緊張感 |
|--------|---------|---------|-------|
| 100〜75% | 序盤 | 依頼受注、採取開始 | 低 |
| 75〜50% | 前半 | 安定して依頼達成 | 中 |
| 50〜25% | 後半 | デッキ調整、効率化 | 高 |
| 25〜0% | 終盤 | 昇格試験に向けた準備 | 最大 |

### 緊張感カーブ

- **Gランク**: チュートリアル、余裕を持って学習
- **F-Eランク**: 基本ループに慣れる
- **D-Cランク**: 品質と期限の厳しさが増す
- **B-Aランク**: 高難度依頼、シビアな判断
- **Sランク**: 最終試験、降格リスクあり

## 5. ゲームサイクル

### メインループ

```
┌──────────────────────────────────────────────────┐
│                    状況把握                       │
│  ランクHP・手札・素材・消耗度・受注依頼を確認    │
└──────────────────────────────────────────────────┘
                    ↓
    ┌────────────────────────────────────┐
    │  今日の行動ポイント(3)で何をする？  │
    └────────────────────────────────────┘
      ↓         ↓         ↓         ↓         ↓
   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐
   │採取地│  │レシピ│  │依頼 │  │依頼 │  │買物 │
   │カード│  │カード│  │受注 │  │納品 │  │    │
   └─────┘  └─────┘  └─────┘  └─────┘  └─────┘
      │         │         │         │         │
      ↓         ↓         ↓         ↓         ↓
   素材獲得   素材消費   期限確認   貢献度   カード/
   (1-3コスト) 調合完成  計画立案   獲得    素材購入
              (1-2コスト)  (0)    (ランクにダメージ)
                    ↓
          ターン経過（消耗度変動、依頼期限減少）
                    ↓
         ランクHP0？ → 昇格試験へ！
                    ↓
              次のターンへ
```

### 状況に応じた判断の例

| 状況 | 優先すべき行動 | 理由 |
|------|--------------|------|
| ゲーム開始時 | 依頼受注 | 目標設定、計画を立てる |
| 素材が足りない | 採取地カード使用 | 素材がないと調合できない |
| 調合可能 | レシピカード使用 | 依頼達成に近づく |
| 完成品がある | 依頼納品 | 貢献度獲得 → ランクにダメージ |
| 消耗度が高い | 回復系依頼を優先 | 消耗度DOWN |
| デッキが弱い | 買い物 | 新カード獲得 |
| 手札が偏っている | 休憩 | 手札入れ替え |

## 7. v4.0 主要変更点

| 要素 | v3.1 | v4.0 |
|------|------|------|
| キャッチコピー | 「受けて、採って、混ぜて、届けろ。」 | 「**依頼を受けて、素材を採って、調合して、ランクを上げろ。**」 |
| 目標 | 開拓度100%×3 | **ギルドランクHP削り→昇格試験→ランクアップ** |
| 敵 | 消耗度（自傷） | **ギルドランク（外敵）** |
| デッキ構成 | 5系統カード | **採取地カード＋レシピカードの2系統** |
| 素材管理 | カードとして扱う | **インベントリ管理（消費制）** |
| 依頼 | 納品で効果発動 | **納品で貢献度（ダメージ）獲得** |
| 進行感 | 開拓地クリア | **ランクアップ（G→S）** |
| 達成感 | バーを埋める | **敵を倒す、昇格試験クリア** |

### v3.1 からの変更理由

- **戦闘感の強化**: 「敵を倒す」という明確な目標
- **達成感の明確化**: ランクアップという区切り
- **成長の可視化**: 解放要素（採取地・レシピ）による進行感
- **デッキの意味強化**: 採取地＝行ける場所、レシピ＝作れるもの
