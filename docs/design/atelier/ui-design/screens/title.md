## 状態遷移

### 初期状態
- BGM再生開始（タイトルBGM）
- 背景アニメーション開始
- セーブデータの存在確認
- セーブデータがない場合、「コンティニュー」ボタンをグレーアウト

### ローディング状態
- 該当なし（BootSceneで処理済み）

### 操作可能状態
- 全ボタンがクリック/キーボード操作可能
- フォーカス移動可能（↑↓キー）

## アニメーション

| トリガー | アニメーション | 時間 | イージング |
|----------|---------------|------|-----------|
| 画面表示時 | タイトルロゴフェードイン | 0.5s | ease-out |
| 画面表示時 | ボタン順次フェードイン | 0.3s × 4 | ease-out |
| ボタンホバー | スケール拡大 (1.05) | 0.1s | ease-in-out |
| ボタン押下 | スケール縮小 (0.95) | 0.05s | linear |
| 画面遷移時 | フェードアウト | 0.3s | ease-in |

---

## イベント

| イベント名 | トリガー | 処理内容 |
|-----------|----------|----------|
| OnNewGameClicked | btn-newgame押下 | スタイル選択画面へ遷移 |
| OnContinueClicked | btn-continue押下 | セーブスロット選択ダイアログ表示 |
| OnSettingsClicked | btn-settings押下 | 設定画面をオーバーレイ表示 |
| OnExitClicked | btn-exit押下 | 終了確認ダイアログ表示 |
| OnNavigateUp | ↑キー押下 | フォーカスを上のボタンへ移動 |
| OnNavigateDown | ↓キー押下 | フォーカスを下のボタンへ移動 |
| OnConfirm | Enter/Space押下 | フォーカス中のボタンを実行 |

---

## アクセシビリティ

- [x] キーボード操作対応（↑↓でフォーカス移動、Enter/Spaceで決定）
- [x] フォーカス表示（選択中ボタンに枠線）
- [x] 効果音フィードバック（ボタン選択SE、決定SE）
