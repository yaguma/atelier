### MainScene レイアウト計算

```
全体: 1280 x 720

┌─────────────────────────────────────────────────────┐ y=0
│  HUDBar (x=200, w=1080, h=56)                      │
│  ※ HEADER_HEIGHT=60 のうち下4pxはスペーサー          │ y=56
├─────────────────────────────────────────────────────┤ y=60
│  PhaseRail (x=200, w=1080, h=48)                   │
├───────────┬──────────────────────────┬──────────────┤ y=108
│           │                          │ ContextPanel │
│ Sidebar   │ ContentContainer         │ (w=260)      │
│ (w=200)   │ x=200                    │ 中心x=1142   │
│ h=660     │ y=108                    │ h=476        │
│           │ 利用可能幅=820           │              │
│           │ (1080-260=820)           │              │
│           │ h=492                    │              │
├───────────┴──────────────────────────┴──────────────┤ y=600
│  FooterUI (x=200, w=824※, h=120)                    │
│  ※FooterUI幅は1024-200=824のハードコード値            │
└─────────────────────────────────────────────────────┘ y=720

ContentContainer計算:
  y = HEADER_HEIGHT(60) + PHASE_RAIL_HEIGHT(48) = 108
  h = 720 - 60 - 48 - 120 = 492
  w = 1280 - 200 = 1080（ContextPanel含む）
  実質作業幅 = 1080 - 260 = 820（ContextPanel分を除く）

ContextPanel計算（中心基準配置）:
  centerX = 1280 - 260/2 - 8 = 1142
  h = 720 - 60 - 48 - 120 - 8*2 = 476（padding込み）
  centerY = 108 + 8 + 476/2 = 354
  実矩形 = x:1012〜1272, y:116〜592
```

### 3.4 操作・イベント

| No. | 操作 | トリガー | 条件 | 処理 | 成功時 | 失敗時 |
|-----|------|---------|------|------|--------|--------|
| ① | 新規ゲーム開始 | ③ボタン押下 / 1キー / Enter | - | セーブデータ有の場合は確認ダイアログ表示 | SCN-002 メインシーンへ遷移（初期状態） | キャンセル時は何もしない |
| ② | コンティニュー | ④ボタン押下 / 2キー | セーブデータが存在 | セーブデータ読み込み・状態復元 | SCN-002 メインシーンへ遷移（復元状態） | 読み込みエラー時はエラーダイアログ表示 |
| ③ | 設定を開く | ⑤ボタン押下 / 3キー | - | 設定ダイアログを表示 | ダイアログ表示 | - |
| ④ | メニュー選択移動 | 上下矢印キー | - | フォーカスを上下に移動 | 選択中ボタンのハイライト | - |
| ⑤ | メニュー決定 | Enter / Space | ボタンにフォーカス中 | フォーカス中のボタンの処理を実行 | 各ボタンに応じた処理 | - |

### 3.5 補足事項

- **キーボード操作**: 上下矢印でメニュー選択、Enter/Spaceで決定、1/2/3キーで直接選択
- **セーブデータ確認**: コンティニューボタンはlocalStorageにセーブデータがない場合は無効（グレーアウト+クリック不可）
- **確認ダイアログ**: 既存セーブデータがある状態で「はじめから」を選択した場合、「既存のセーブデータが上書きされます。よろしいですか？」と確認
