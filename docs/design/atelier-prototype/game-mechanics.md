# ゲームメカニクス設計

## 状態設計

### リソース一覧

| リソース | 初期値 | 最大値 | 役割 |
|---------|-------|-------|------|
| デッキ残量 | 15枚 | - | 生存条件: 0枚でゲームオーバー |
| エネルギー | 3 | 10 | 行動制限: 毎ターン3回復 |
| 安定度 | 10 | 10 | 暴発条件: 0以下で失敗 |
| 手札 | 3枚 | 3枚 | 毎ターン3枚まで補充 |

### 依頼（3件同時）

| 依頼 | 達成条件 | 難易度 |
|------|---------|--------|
| 依頼1: 回復薬 | 火 >= 10 | 簡単 |
| 依頼2: 解毒剤 | 水 >= 8 かつ 火 >= 5 | 中 |
| 依頼3: 魔力薬 | 火 >= 15 かつ 品質 >= 3 | 難 |

## カード設計（5種類）

### 1. 火の鉱石（安全カード）

```
名前：火の鉱石
コスト：1エネルギー
即時効果：火属性 +3
将来への影響：なし（安定した進捗）
リスク：低い（確実だが効率は普通）
カテゴリ：安全
```

### 2. 不安定な結晶（危険カード）

```
名前：不安定な結晶
コスト：2エネルギー
即時効果：火属性 +8、安定度 -3
将来への影響：暴発リスク（安定度0以下で失敗）
リスク：高い（高効率だが暴発の危険）
カテゴリ：危険
```

### 3. 安定剤（準備カード）

```
名前：安定剤
コスト：1エネルギー
即時効果：安定度 +5
将来への影響：後の危険カード使用を可能に
リスク：単体では進捗ゼロ（在庫を消費するだけ）
カテゴリ：準備
```

### 4. 触媒の粉（消費カード）

```
名前：触媒の粉
コスト：0エネルギー
即時効果：次に使うカードの属性値2倍
将来への影響：コンボ前提、単体では無価値
リスク：組み合わせミスで在庫無駄遣い
カテゴリ：消費
```

### 5. 水の草（条件付きカード）

```
名前：水の草
コスト：1エネルギー
即時効果：水属性 +3、火属性が5以上なら追加+2
将来への影響：順序依存（火を先に貯めると有利）
リスク：順序を間違えると効率低下
カテゴリ：条件
```

## 初期デッキ構成（15枚）

| カード | 枚数 | 役割 |
|--------|------|------|
| 火の鉱石 | 5枚 | 安定した火属性供給 |
| 不安定な結晶 | 3枚 | 高効率だがリスキー |
| 安定剤 | 3枚 | リスク管理 |
| 触媒の粉 | 2枚 | コンボ用 |
| 水の草 | 2枚 | 水属性供給 |

## ターン構造

### ターン開始
1. エネルギー +3（最大10まで）
2. 手札を3枚まで補充（デッキから引く = 在庫減少）

### プレイヤーアクション
- カードをプレイ（エネルギー消費、カード完全消滅）
- または「ターン終了」を選択

### ターン終了
- 手札は維持（捨て札なし = シンプル化）
- 依頼達成判定
- ゲームオーバー判定

---

## 勝敗条件

### 勝利条件

3つの依頼をすべて達成する：

1. **依頼1達成**: 火属性の累計が10以上
2. **依頼2達成**: 水属性の累計が8以上 かつ 火属性が5以上
3. **依頼3達成**: 火属性の累計が15以上 かつ 品質が3以上

### 敗北条件

以下のいずれかで即ゲームオーバー：

1. **在庫切れ**: デッキ残量が0枚になる
2. **暴発**: 安定度が0以下の状態でカードをプレイ

---

## 計算式

### 属性値計算

```
現在の属性値 = 累計プレイカードの属性値合計 × 触媒ボーナス
触媒ボーナス = 触媒使用中なら2倍、それ以外は1倍
```

### 依頼達成判定

```
if (火属性 >= 依頼.必要火属性 && 水属性 >= 依頼.必要水属性 && 品質 >= 依頼.必要品質) {
    依頼達成
    報酬: デッキにカード+2枚追加
}
```

### 暴発判定

```
if (安定度 <= 0) {
    カードプレイ不可（暴発 = ゲームオーバー）
}
```

## バランス設計の意図

### 在庫消費ペース

- 初期デッキ15枚
- 各依頼に平均4〜5枚使用を想定
- 無駄遣いすると3つ目の依頼で在庫切れ

### 難易度曲線

- 依頼1: ほぼ確実にクリア可能（チュートリアル的）
- 依頼2: 適切なカード選択が必要
- 依頼3: 効率的なプレイと報酬獲得が必須

### リプレイ動機

| 要因 | 設計対応 |
|------|---------|
| 「あのカードを温存すれば...」 | 明確な失敗原因がわかる |
| 「危険カードをもっと使えば効率的」 | 上達余地が見える |
| 「依頼の順番を変えてみよう」 | 戦略の多様性 |
